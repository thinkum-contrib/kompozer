Liste des menus de &brandShortName;

Cette liste décrit tous les menus de la fenêtre principale de &brandShortName;.

Fichier

Nouveau

Ouvre une nouvelle fenêtre de &brandShortName;.

Ouvrir un fichier…

Affiche la boîte de dialogue Ouvrir un fichier vous permettant de sélectionner un fichier depuis votre disque dur ou du réseau. Avec &brandShortName; vous pouvez voir les fichiers de type HTML, modèles HTML, et texte.

Ouvrir une page Web…

Affiche la boîte de dialogue Ouvrir une page Web vous permettant de saisir l'URL du document ou de choisir un fichier local à ouvrir.

Pages récentes

Affiche la liste des fichiers récemment édités.

Fermer

Ferme la fenêtre courante.

Fermer tout

Ferme tous les onglets ouverts dans la fenêtre courante.

Enregistrer

Enregistre le fichier ouvert dans l'onglet actif.

Enregistrer sous…

Enregistre le fichier que vous éditez en vous proposant de lui donner un nom de fichier différent.

Enregistrer et modifier l'encodage…

Vous permet de modifier l'encodage de la page avant d'afficher la boîte de dialogue Enregistrer sous.

Publier

Ouvre la boîte de dialogue Publier vous permettant de publier la page en cours d'édition.

Publier sous…

Ouvre la boîte de dialogue Publier vous permettant de publier la page en cours d'édition. Se comporte comme Publier !

Restaurer

Annule tous les changements effectués sur le document actuel depuis la dernière sauvegarde. Revient à la dernière sauvegarde du document.

Exporter au format texte…

Exporte le contenu du document actuel au format texte brut.

Aperçu dans le navigateur

Affiche la page actuelle dans le navigateur par défaut.

Mise en Page…

Ouvre la boîte de dialogue de mise en page pour configurer les options d'impression.

Aperçu avant impression

Ouvre une fenêtre montrant l'aspect qu'aura votre document quand vous l'imprimerez sur du papier.

Imprimer…

Affiche la boîte de dialogue Imprimer où vous pouvez spécifier le nombre de pages à imprimer, etc. Cliquez sur OK pour imprimer la page.

Quitter

Ferme toutes les fenêtres de &brandShortName; incluant toutes celles avec plus d'un onglet ouvert, et quitte &brandShortName; complètement. Si vous avez des documents non enregistrés, &brandShortName; vous demandera de confirmer votre choix pour empêcher toute sortie accidentelle.

Édition

Annuler

Annule la dernière action d'édition effectuée dans le document actuel. Si vous décidez plus tard que vous ne vouliez pas défaire une action, employez la commande Refaire.

Refaire

Refait la dernière action annulée.

Couper

Coupe le texte sélectionné depuis le document et le stocke dans le presse-papiers. Collez le texte ailleurs en utilisant la commande Coller.

Copier

Copie le texte sélectionné dans le presse-papiers. Collez le texte ailleurs en utilisant la commande Coller.

Coller

Colle le texte issu du presse-papiers par la commande Couper ou Copier à la position actuelle du curseur.

Coller sans mise en forme

Colle le texte issu du presse-papiers par la commande Couper ou Copier à la position actuelle du curseur comme du texte simple c'est-à-dire que les informations de mise en page du texte du presse-papiers sont ignorées.

Effacer

Efface le texte sélectionné dans le document actuel.

Tout sélectionner

Sélectionne tout le texte dans le document actuel.

Rechercher et remplacer…

Affiche la boîte de dialogue où vous pouvez saisir une chaîne de texte qui sera recherchée et remplacée par une autre chaîne de texte. La recherche est effectuée sur la page courante.

Rechercher à nouveau

Recherche l'occurrence suivante du texte saisi dans la boîte de dialogue Rechercher et remplacer…. La recherche est effectuée sur la page courante.

Rechercher le précédent

Recherche l'occurrence précédente du texte saisi dans la boîte de dialogue Rechercher et remplacer….

Vérifier l'orthographe

Ouvre le correcteur orthographique et commence la correction orthographique sur le texte de la page courante.

Dissocier du modèle

Dissocie le document actuellement édité du modèle pour l'enregistrer comme une page html.

Paramètres des sites de publication

Ouvre le gestionnaire de sites pour configurer ou changer les paramètres pour le site de publication.

Affichage

Barre d'outils

Principale

Affiche ou cache la barre d'outils de composition, qui contient habituellement les boutons standard de composition (Ancre, Lien, Image, etc).

Mise en forme

Affiche ou cache la barre d'outils de mise en forme, qui contient divers boutons de mise en forme.

Modes d'édition

Affiche ou cache la barre d'outils de mode d'édition. Cette barre d'outils se trouve dans la fenêtre de &brandShortName; juste au-dessus de la barre d'état. Cette barre vous permet de basculer entre les différentes vues comme Balises HTML, Source HTML, etc.

Barre d'état

Affiche ou cache la barre d'état située en bas de la fenêtre de &brandShortName;.

Règles

Affiche ou cache les règles horizontales et verticales.

Personnalisation des barres d'outils

Barre d'outils principale

Ouvre un panneau pour modifier la barre d'outils principale. Vous pouvez glisser et déposer des boutons depuis le panneau vers la barre d'outils principale pour ajouter de nouveaux boutons ou inversement pour retirer des boutons.

Barre d'outils de mise en forme

Ouvre un panneau pour modifier la barre d'outils de mise en forme. Vous pouvez glisser et déposer des boutons depuis le panneau vers la barre d'outils de mise en forme pour ajouter des nouveaux boutons ou inversement pour retirer des boutons.

Gestionnaire de sites

Escamote le panneau du Gestionnaire de sites dans la fenêtre de &brandShortName;.

Mode d'édition normal

Bascule vers le mode d'édition normal.

Balises HTML

Bascule vers le mode affichant les balises HTML.

Source HTML

Bascule vers le mode affichant le code source HTML.

Aperçu

Bascule vers le mode d'aperçu.

Retour à la ligne

Affiche ou cache les retours à la ligne.

Mise en évidence des blocs

Affiche ou cache le contour des éléments de bloc dans la page.

Taille du texte

Plus petit

Affiche le texte une taille en dessous dans la page Web. Vous pouvez également faire ceci avec la souris (voir Plus grand pour des instructions). Pour restaurer la taille de texte par défaut, utilisez Ctrl+0.

Plus grand

Affiche le texte une taille au-dessus dans la page Web. Vous pouvez également faire ceci avec la souris en maintenant la touche Ctrl enfoncée tout en utilisant la molette ou en pressant + ou - pour augmenter ou diminuer la taille du texte. Pour restaurer la taille de texte par défaut, utilisez Ctrl+0.

Encodage des caractères

Vous permet de changer manuellement l'encodage des caractères sur une page Web. &brandShortName; fait habituellement ceci automatiquement.

Insertion

Le menu d'insertion vous permet d'insérer divers éléments HTML dans le document en cours d'édition.

Image…

Ouvre la boîte de dialogue Propriétés de l'image pour insérer une image dans le document actuel.

Tableau…

Ouvre la boîte de dialogue Insérer un tableau pour insérer un tableau dans le document actuel.

Formulaire

Lien…

Ouvre la boîte de dialogue Propriétés du lien pour créer un lien dans le document actuel. Si du texte est sélectionné lorsque vous utilisez cette commande, le texte sélectionné sera transformé en lien. Vous pouvez définir la cible du lien et les informations de relation (XFN) en cliquant sur Plus de propriétés.

Ancre…

Ouvre la boîte de dialogue Propriétés de l'ancre pour saisir le nom de l'ancre à insérer à la position du curseur.

Ligne horizontale

Insère une ligne horizontale dans le document.

Code HTML…

Ouvre la boîte de dialogue Insérer du code HTML pour insérer manuellement du code HTML. Vous pouvez ainsi insérer des scripts, des balises inconnues de &brandShortName;, etc.

Caractères et symboles…

Ouvre la boîte de dialogue Insérer un caractère pour insérer des lettres accentuées ou des symboles usuels dans le document.

Table des matières…

Ouvre la boîte de dialogue Table des matières pour insérer une table des matières à l'emplacement du curseur. La table peut être établie à partir des titres de niveaux 1 à 6, des paragraphes ou des balises <div>.

Autres insertions

Insère un Smart Widgets dans votre document.

Bouton Made With &brandShortName;

Insère un bouton Made with &brandShortName; dans votre document.

Calendrier

Insère un calendrier dans votre document.

Bouton Fait avec &brandShortName;

Insère un bouton Fait avec &brandShortName; dans votre document.

Modèles

Quand vous travaillez sur un modèle, vous pouvez utiliser ce menu pour insérer une zone éditable dans votre modèle.

Code PHP…

Ouvre la boîte de dialogue Insérer du code PHP pour insérer du code PHP.

Commentaires…

Ouvre la boîte de dialogue Insérer un commentaire pour insérer un commentaire HTML.

Saut de lignes sous les images

Insère un saut de ligne sous les images pour que le texte saisi à côté de l'image apparaisse en dessous.

Format

Ce menu contient diverses options pour mettre en forme le texte du document.

Police

Vous pouvez choisir la police à appliquer sur le texte sélectionné dans le document en cours. Vous pouvez choisir la police désirée dans la liste contenant toutes les polices disponibles de votre système.

Taille

En sélectionnant Plus petite ou Plus grande, vous pouvez changer la taille du texte visionné dans &brandShortName;.

Sélectionner Très petite, Petite, Moyenne, Grande, Très grande ou Extra-grande change la taille du texte sélectionné dans le document en cours.

Style du Texte

Vous pouvez sélectionner le style de texte HTML qui sera appliqué au texte sélectionné dans le document.

Gras

Affiche le texte sélectionné en caractères gras. Ex. : texte en caractères gras

Italique

Affiche le texte sélectionné en italique. Ex. : texte en italique

Souligné

Souligne le texte sélectionné. Ex. : texte souligné

Barré

Barre le texte sélectionné. Ex. : texte barré

Exposant

Met le texte sélectionné en exposant. Ex. : texte en exposant.

Indice

Met le texte sélectionné en indice. Ex. : texte en indice.

Largeur fixe

Affiche le texte sélectionné en largeur fixe. Ex. : texte en largeur fixe.

Insécable

Le code HTML convertit les espaces multiples en une seule. Pour marquer une espace insécable, utilisez cette option après avoir sélectionné l'espace dans le document. Ex. : il y a trois espaces insécables   ici.

Mise en évidence

Met le texte en évidence. Ex. : texte mis en évidence

Mise en évidence prononcée

Met le texte en évidence de manière prononcée. Ex. : texte mis en très forte évidence.

Citation

Définit le texte comme une citation. Ex. : Citation

Abréviation

Indique que le texte est une abréviation. Ex. : Mlle

Acronyme

Indique que le texte est un acronyme. Ex. : ONU

Code

Indique que le texte sélectionné représente du code informatique. Ex. : code informatique

Brouillon

Indique que le texte sélectionné est une sortie de programme informatique. Ex. : échantillon de code

Variable

Indique que le texte sélectionné est une variable. Ex. : variable

Couleur du texte…

Ouvre la fenêtre de sélection de couleur pour choisir celle du texte sélectionné.

Sens d'écriture

Permet de choisir le sens d'écriture de gauche à droite ou de droite à gauche. Le texte sélectionné doit faire partie d'un bloc et ne peut être une sélection dans un bloc.

Annuler le style de texte

Annule l'application de tout style de texte utilisé à partir de la position courante du curseur.

Annuler le lien

Indique que le texte n'est plus un lien à partir de la position courante du curseur.

Supprimer les ancres

Supprime l'ancre sélectionnée.

Paragraphe

Permet de marquer le texte sélectionné comme étant du corps de texte, un paragraphe, un titre de niveau 1 à 6, une adresse, du texte préformaté ou un conteneur générique (div).

Liste

Permet de créer une liste avec une option pour divers types de listes : Aucune, Puces, Numérotation, Terme ou Définition. Choisir Propriétés de la liste ouvre une boîte de dialogue pour éditer manuellement les propriétés de la liste.

Augmenter le retrait

Augmente le retrait du bloc de texte sélectionné.

Diminuer le retrait

Diminue le retrait du bloc de texte sélectionné.

Aligner

Vous permet de choisir l'alignement du texte sélectionné. Vous pouvez centrer le texte, l'aligner à gauche ou à droite ou simplement le justifier.

Grille magnétique…

Ouvre une boîte de dialogue pour définir la résolution en pixels pour déplacer une couche.

Propriétés avancées…

Ouvre l'Éditeur de propriétés avancées où vous pouvez éditer/ajouter manuellement des attributs HTML, des styles ou même des événements JavaScript.

Couleurs et fond de page…

Ouvre une fenêtre pour sélectionner la couleur et le fond de page. Les utilisateurs peuvent sélectionner la couleur pour le texte normal, le texte des liens, le texte des liens actifs et des liens visités. Vous pouvez également choisir la couleur de fond de page et si vous le désirez, vous pouvez aussi ajouter une image de fond pour votre page Web.

Titre et propriétés de la page…

Ouvre la fenêtre Propriétés de la page où vous pouvez ajouter le titre, l'auteur et la description pour votre page Web. Vous pouvez également spécifier la langue, le sens d'écriture et l'encodage que vous utiliserez pour la page en cours. Pour utiliser cette page en tant que modèle, cochez la case Cette page est un modèle.

Tableau

Ce menu vous permet de créer/éditer des tableaux dans un document HTML.

Insérer

Ouvre un sous-menu avec diverses options pour créer/éditer des tableaux.

Tableau…

Ouvre une boîte de dialogue pour choisir diverses propriétés du tableau qui sera inséré à l'endroit de la position courante du curseur. Les utilisateurs peuvent choisir la taille du tableau en cliquant sur la grille de tableau affichée ou en choisissant une taille de tableau précise. D'autres propriétés de cellule comme cellspacing, cellpadding, etc. peuvent être aussi définies.

Ligne au-dessus

Insère une ligne vide au-dessus de la ligne active courante.

Ligne en dessous

Insère une ligne vide en dessous de la ligne active courante.

Colonne avant

Insère une colonne vide avant la colonne active courante.

Colonne après

Insère une colonne vide après la colonne active courante.

Cellule avant

Insère une cellule vide avant la cellule active courante.

Cellule après

Insère une cellule vide après la cellule active courante.

Sélectionner

Sélectionne divers composants du tableau actif courant pour les éditer.

Tableau

Sélectionne le tableau entier.

Ligne

Sélectionne la ligne courante active.

Colonne

Sélectionne la colonne courante active.

Cellule

Sélectionne la cellule courante active.

Toutes les cellules

Sélectionne toutes les cellules. Ceci est différent de la sélection du tableau en entier, car les modifications seront appliquées individuellement à toutes cellules plutôt qu'au tableau.

Supprimer

Ouvre un sous-menu pour supprimer divers composants de tableau.

Tableau

Supprime le tableau actif courant. À utiliser avec précaution : toutes les données du tableau seront perdues.

Ligne(s)

Supprime la ligne active courante ou toutes les lignes sélectionnées du tableau.

Colonne(s)

Supprime la colonne active courante ou toutes les colonnes sélectionnées du tableau.

Cellule(s)

Supprime la cellule active courante ou toutes les cellules sélectionnées du tableau.

Contenu de cellules

Supprime le contenu de la cellule active.

Fusionner avec la cellule suivante

Fusionne la cellule courante du tableau avec la cellule qui la suit immédiatement.

Scinder la cellule

Scinde une cellule qui a une valeur colspan supérieure à 1.

Créer un tableau à partir de la sélection

Crée un tableau à partir du texte sélectionné. Une ligne est créée pour chaque paragraphe du texte sélectionné. Les séparations de colonnes peuvent être choisies : une virgule (,), une espace ( ) ou un caractère défini par l'utilisateur.

Couleur de fond de tableau ou de cellule…

Ouvre une boîte de dialogue pour éditer la couleur de fond du tableau ou de la cellule.

Propriétés du tableau…

Ouvre la fenêtre d'édition des propriétés de tableau et de cellules. Vous pouvez changer toutes les propriétés de tableau et de cellule en utilisant cette simple interface graphique.
