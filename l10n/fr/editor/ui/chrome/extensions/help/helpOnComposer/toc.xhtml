Ajouter une table des matières à votre page Web

Pour un document html ayant différentes sections, c'est-à-dire un article avec des sections et des sous-sections, et des titres pour chaque section et sous-section, il devient difficile de naviguer si tout le texte est d'un seul tenant. Pour mettre en œuvre une navigation facile dans les différentes sections, une table des matières peut être générée facilement avec &brandShortName;. Pour générer une table des matières pour un document html :

  1. Utilisez les balises de titres html (h1, h2, h3, h4, h5, et h6) pour libeller les différents niveaux de sections et sous-sections du document. Puis enregistrez le document.
  2. Cliquez sur Insertion > Table des matières et choisissez Insérer. La fenêtre Table des matières apparaîtra.
  3. Sélectionnez les niveaux de titres utilisés pour les différentes sections et sous-sections dans le document.
  4. Cliquez sur OK pour construire la table des matières.

La table des matières générée par défaut montre toutes les entrées numérotées et des liens pointent vers les ancres des sections respectives. En décochant Numéroter toutes les entrées dans la table des matières, l'affichage se fera sous forme de liste à puces. En cochant Table des matières en lecture seule, les sections seront affichées sous forme de texte simple, sans ancres.

Mettre à jour ou supprimer une table des matières

Si le document est modifié pour y ajouter plus de sections et de sous-sections après avoir construit la table des matières, elle ne contiendra pas les nouvelles sections et sous-sections. Pour ajouter ces nouvelles entrées, cliquez sur Insertion puis Table des matières et choisissez Mettre à jour. La table des matières sera mise à jour pour afficher les nouvelles entrées.

Pour supprimer toute la table des matières, cliquez sur Insertion, puis Table des matières et choisissez Supprimer. La table sera supprimée et le code source html sera automatiquement nettoyé de tout code relatif à la table des matières.
