Créer des pages Web avec &brandShortName;

&brandShortName; vous permet de créer vos propres pages Web, puis de les publier sur Internet. Vous n'avez pas besoin de connaître le language HTML pour l'utiliser. Il est aussi facile de prise en main qu'un logiciel de traitement de texte.

Avec les boutons de la barre d'outils, vous pourrez facilement ajouter des listes, des tableaux, des images, des liens vers d'autres pages, de la couleur… Vous verrez à quoi ressemblera votre document sur Internet au fur et à mesure que vous le réaliserez et vous pourrez aisément le partager avec d'autres personnes, quel que soit le navigateur ou le client de messagerie HTML qu'ils utilisent.

Commencer une nouvelle page

Créer une nouvelle page

&brandShortName; est un éditeur HTML permettant de créer et de modifier des pages Web. Éditeur WYSIWYG, le composeur permet de voir exactement la façon dont la page sera affichée à mesure de sa création. Il n'est pas nécessaire de connaître le langage HTML, dans la mesure où la plupart des fonctions HTML sont disponibles sous la forme de commandes dans les barres d'outils et les menus.

&brandShortName; vous permet également d'éditer le code source HTML si vous le désirez. Pour voir ou éditer le code source HTML, ouvrez le menu Affichage, et choisissez Source HTML, ou cliquez sur l'onglet <HTML> Source dans la barre d'outils Modes d'édition au bas de la fenêtre de &brandShortName;.

Pour créer une page Web, procédez comme suit. Une fois que vous avez entamé une page, vous pouvez ajouter et modifier du texte tout comme dans un traitement de texte.

Pour créer une nouvelle page dans &brandShortName; :

  • Cliquez sur le bouton Nouveau dans la barre d'outils de &brandShortName;.

Pour commencer à partir d'un fichier HTML stocké sur votre disque dur :

  1. Ouvrez &brandShortName;. Vous verrez alors la fenêtre &brandShortName;.
  2. Ouvrez le menu Fichier et choisissez Ouvrir un fichier. La boîte de dialogue Ouvrir un fichier HTML apparaîtra.
  3. Sur votre disque dur, sélectionnez le fichier que vous voulez modifier.
  4. Cliquez sur Ouvrir pour afficher le fichier spécifié dans une fenêtre &brandShortName;.

Pour modifier une page Web :

  1. Ouvrez &brandShortName;.
  2. Cliquez sur le menu Fichier et choisissez Ouvrir une page Web. Vous verrez apparaître la boîte de dialogue Ouvrir une page Web.
  3. Saisissez l'adresse URL de la page à éditer (par exemple, http://www.mozilla.org) et cliquez sur le bouton Ouvrir. Vous verrez alors la page s'afficher dans &brandShortName;.

&brandShortName; vous offre également la possibilité de créer des documents utilisant le format XHTML. Pour créer un document XHTML, cochez l'option créer un document XHTML dans la boîte de dialogue Créer un nouveau document ou un modèle.

Conseil : dans la fenêtre de &brandShortName; vous pouvez rapidement ouvrir les derniers fichiers sur lesquels vous avez travaillé en ouvrant le menu Fichier, en choisissant Pages récentes puis le fichier que vous désirez dans la liste.

Onglets - Édition multi-documents

&brandShortName; vous permet d'éditer plusieurs documents à la fois. Au lieu de basculer d'une fenêtre à l'autre pour éditer plus d'un document, &brandShortName; utilise des onglets. Plusieurs documents peuvent être ouverts en même temps avec &brandShortName; dans la même fenêtre, mais dans des onglets différents.

Les onglets peuvent être ouvert soit en utilisant Ctrl+T, soit en choisissant Créer dans : un nouvel onglet dans la boîte de dialogue Ouverture d'un nouveau fichier. Chaque onglet est indépendant et l'état de chacun est indiqué, c'est-à-dire que s'il est modifié et n'a pas été enregistré, une disquette rouge apparaît dans l'onglet correspondant.

Enregistrer et tester votre nouvelle page

Vous pouvez enregistrer des documents créés avec &brandShortName; au format HTML et texte uniquement. L'enregistrement d'un document au format HTML conserve sa mise en forme, notamment les styles de texte (par exemple, gras ou italique), les tableaux, les liens et les images. L'enregistrement au format texte uniquement supprime toutes les balises HTML mais conserve le texte.

Pour enregistrer un document au format HTML :

  • Ouvrez le menu Fichier et choisissez Enregistrer ou choisissez le bouton correspondant dans la barre d'outils.

    Si vous n'avez pas encore donné de titre à la page, &brandShortName; vous invite à le faire. Le titre s'affiche dans la barre de titre de la fenêtre du navigateur lorsque vous y affichez la page. Il apparaît également dans votre liste de marque-pages, le cas échéant.

    &brandShortName; vous invite ensuite à choisir le chemin d'enregistrement. Veillez à conserver l'extension .html du nom de fichier. Quand vous enregistrez ou éditez des documents XHTML, l'extension du fichier devrait être .xhtml. Les fichiers avec l'extension .html mais avec un DOCTYPE XHTML ne sont pas considérés comme des documents xhtml.

Pour changer le nom ou l'emplacement d'un fichier HTML existant :

  • Choisissez Fichier > Enregistrer sous pour attribuer à la page un autre nom de fichier ou un autre emplacement.

Lorsque vous enregistrez une page dans &brandShortName;, tous ses éléments (le code HTML, les images et les autres fichiers tels que des fichiers son et des feuilles de style), sont enregistrés localement sur votre disque dur. Si vous désirez sauvegarder uniquement le code HTML, vous devez modifier les préférences de &brandShortName; concernant la sauvegarde des pages. Pour plus d'informations sur la façon de modifier les préférences d'enregistrement, voir la rubrique Préférences de &brandShortName;.

Si le chemin d'une image est absolu (commençant par http://) et que vous êtes connecté à Internet, vous verrez toujours cette image dans le document depuis les fenêtres de &brandShortName; et du navigateur. Cependant, si l'adresse de l'image est relative par rapport à celui de la page (commençant par file:///), alors vous ne verrez pas l'image dans la version locale du document.

Pour enregistrer un document au format texte uniquement :

  1. Ouvrez le menu Fichier et choisissez Exporter au format texte.
  2. Entrez le nom du fichier et son emplacement d'enregistrement.

Remarque : les images ne s'affichent pas dans les documents enregistrés au format texte.

Conseil : vous pouvez aussi choisir Restaurer dans le menu Fichier pour récupérer la dernière version enregistrée du document sur lequel vous travaillez. N'oubliez pas que les modifications en cours seront perdues.

Pour afficher la page dans une fenêtre du navigateur afin de tester les liens :

  • Ouvrez le menu Fichier et sélectionnez Aperçu dans le navigateur (ou cliquez sur le bouton Navigateur dans la barre d'outils). Si vous n'avez pas encore enregistré le document, &brandShortName; vous invite à entrer un titre de page, un nom de fichier et un emplacement. La fenêtre de &brandShortName; reste ouverte en arrière-plan de la nouvelle fenêtre du navigateur.
