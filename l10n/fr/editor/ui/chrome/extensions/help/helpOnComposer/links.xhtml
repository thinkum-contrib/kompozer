Créer des liens avec &brandShortName;

Créer des liens vers la même page

Pour créer des liens vers la même page, par exemple un lien que le lecteur peut utiliser pour aller de section en section, vous devez créer une ancre (une cible d'emplacement), puis créer un lien qui pointe vers cette ancre. Les ancres sont également appelées ancres nommées.

  1. Cliquez pour placer le curseur au début d'une ligne où vous voulez créer une ancre, ou sélectionnez du texte.
  2. Cliquez sur le bouton Ancre ou ouvrez le menu Insertion puis choisissez Ancre. Vous verrez alors la boîte de dialogue Propriétés de l'ancre.
  3. Tapez un nom unique pour l'ancre dans le champ Nom de l'ancre (jusqu'à 30 caractères). Si vous incluez des espaces, elles seront converties en traits de soulignement ( _ ). Si vous avez sélectionné du texte lors de l'étape 1, cette boîte contiendra déjà un nom.
  4. Cliquez sur OK. Une icône d'ancre apparaîtra dans votre document pour marquer l'emplacement de l'ancre :

Pour créer le lien sur lequel les lecteurs pourront cliquer pour rejoindre l'emplacement de l'ancre :

  1. Sélectionnez le texte ou l'image que vous voulez lier à l'ancre.
  2. Cliquez sur le bouton Lien ou ouvrez le menu Insertion et choisissez Lien. Vous verrez alors la boîte de dialogue Propriétés du lien.
    • Si vous créez un lien vers une page HTML sur votre ordinateur, cliquez sur Parcourir… pour le situer.
    • Si vous créez un lien vers une ancre (une cible), sélectionnez-la dans la liste des ancres disponibles de la page.
    • Si vous créez un lien vers un niveau de titre (par exemple, Titre 1 - Titre 6), sélectionnez-le dans la liste des titres disponibles de la page.
  3. Cliquez sur OK.

Note : Pour tester le lien que vous venez de créer, ouvrez le menu Fichier et choisissez Aperçu dans le navigateur, puis cliquez sur le lien.

Conseil : Si vous n'avez pas créé d'ancre auparavant, vous pouvez utiliser la boîte de dialogue Lien pour créer des liens vers des titres déjà présents sur la page.

Créer des liens vers d'autres pages

Vous pouvez créer des liens de votre page vers des pages de votre ordinateur ou de votre réseau d'entreprise, ou vers des pages sur Internet.

Conseil : Il est préférable d'enregistrer ou de publier vos pages avant d'y insérer des images. Ceci permet à &brandShortName; d'utiliser automatiquement des références relatives pour les images lorsque vous les insérer.

Pour créer des liens vers une autre page :

  1. Cliquez pour placer le curseur à l'endroit où vous voulez créer le lien, ou sélectionnez le texte ou l'image que vous voulez lier à l'ancre.
  2. Cliquez sur le bouton Lien. Vous verrez alors la boîte de dialogue Propriétés du lien.
  3. Définissez votre lien :
    • Lien texte : Si vous avez sélectionné une image ou du texte avant de cliquer sur le bouton Lien, le texte sélectionné ou le nom du fichier image apparaîtra ici. Sinon, vous devrez entrer le texte que vous voulez utiliser pour le lien.
    • Emplacement du lien : Tapez le chemin d'accès et le nom de fichier ou l'URL distante à lier. Si vous n'êtes pas sûr du chemin d'accès et du nom de fichier pour un fichier local, cliquez sur Parcourir… pour le rechercher sur votre disque dur ou le réseau. Pour les URL distantes, vous pouvez copier l'URL à partir du champ adresse de votre navigateur. Vous pouvez aussi sélectionner une ancre ou un titre dans la page que vous voulez lier.
    • L'URL est relative à l'adresse de la page : Si cette option est cochée, &brandShortName; convertit l'URL pour qu'elle soit relative à l'adresse de la page. Ceci est particulièrement utile si vous prévoyez de publier vos pages sur un serveur Web pour que d'autres personnes puissent les voir. Utiliser des URL relatives vous permet de conserver tous vos fichiers liés au même emplacement relatif les uns par rapport aux autres, sans tenir compte de leur emplacement sur votre disque dur ou sur un serveur Web.

      Si vous décochez cette option, &brandShortName; convertir l'URL en une URL complète (absolue). On utilise généralement des URL absolues pour lier des images présentes sur d'autres serveurs Web (pas pour des images stockées localement sur votre disque dur).

      Si vous n'avez pas encore enregistré ou publié la page, vous devez d'abord l'enregistrer pour activer cette option.

  4. Pour appliquer des attributs supplémentaires ou des événements JavaScript, cliquez sur Édition avancée… pour afficher l'Éditeur des propriétés avancées.
  5. Cliquez sur OK.
  6. Pour tester un lien que vous venez de créer, cliquez sur le bouton Navigateur, puis cliquez sur le lien pour vous assurer qu'il fonctionne comme prévu.

Conseil : Vous pouvez copier un lien rapidement en faisant un glisser-déposer de ce lien à partir d'une autre fenêtre et en le déposant sur votre page. Par exemple, vous pouvez faire glisser un lien à partir d'une page Web, d'un marque-page ou d'une fenêtre de courrier et le déposer sur votre page. Vous pouvez également faire un clic droit (Control+click sur Mac OS) sur un lien d'une page Web et sélectionner Copier l'adresse du lien à partir du menu. Puis, vous pouvez copier l'adresse du lien dans le champ Emplacement du lien dans la boîte de dialogue Propriétés du lien.

Ajouter des informations XFN à vos liens

Lorsque vous créez des liens sur votre page Web vers les pages d'autres personnes, &brandShortName; fournit la fonctionnalité d'inclure des informations XFN (XHTML Friends Network). XFN est un moyen simple de représenter les relations entre personnes en utilisant des hyperliens. Pour ajouter des informations XFN :

  1. Faites un clic droit sur le lien auquel vous voulez ajouter des informationsXFN.
  2. Sélectionnez Propriétés du lien….
  3. Dans la fenêtre Propriétés du lien, cliquez sur Plus de propriétés.
  4. Sélectionnez les entrées correspondantes sous la section Relation à ajouter en tant qu'informations XFN.
  5. Cliquez sur OK.
  6. Cliquez sur le bouton Enregistrer pour enregistrer les changements/ajouts dans le document.

Note : Certaines propriétés XFN sont de type bouton radio c'est-à-dire que vous ne pouvez en choisir qu'au plus une (propriété), les autres sont des cases à cocher c'est-à-dire que vous pouvez cocher plusieurs propriétés.

Utiliser des images en tant que liens

Vous pouvez faire se comporter des fichiers images tels que JPEG, GIF, ou PNG comme des liens dans vos pages. Quand le lecteur clique sur une image liée, le navigateur affiche la page vers laquelle l'image est liée.

  1. Sélectionnez une image dans votre page.
  2. Cliquez sur le bouton Lien de la barre d'outils, ou ouvrez le menu Insertion et choisissez Lien.
  3. Utilisez la boîte de dialogue Propriétés du lien pour lier une image vers une ancre ou un titre de la page en cours, ou vers une autre page ou une page distante.

Conseil : Faites un glisser-déposer d'une image liée à partir de la fenêtre de votre navigateur vers une fenêtre de &brandShortName; pour y copier à la fois l'image et le lien.

Note : Pour supprimer la bordure bleue entourant les images utilisées comme liens, ouvrez la boîte de dialogue Propriétés du lien, cliquez sur l'onglet, et décochez l'option Encadrer les images qui portent un lien.

Supprimer ou rompre les liens

Pour supprimer un lien :

  1. Sélectionnez le texte lié (normalement souligné en bleu) ou l'image.
  2. Ouvrez le menu Format et choisissez Supprimer les liens.

Pour rompre un lien, de sorte que le texte que vous tapez après celui-ci, ne soit pas inclus dans le lien :

  1. Cliquez pour placer le curseur à l'endroit où vous voulez que le lien s'arrête.
  2. Ouvrez le menu Format et choisissez Annuler le lien.
