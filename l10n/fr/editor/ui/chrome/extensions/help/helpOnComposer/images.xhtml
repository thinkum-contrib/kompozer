Ajouter des images à votre page Web

Insérer une image à votre page

Vous pouvez insérer des images GIF, JPEG, BMP, and PNG (Portable Network Graphics) dans votre page Web. Vous pouvez également les utiliser pour créer des liens. Quand vous insérer une image, &brandShortName; enregistre une référence à l'image dans votre page.

Note : Si vous prévoyez de publier vos pages sur le Web, il est préférable de ne pas utiliser d'images BMP pour vos pages.

Conseil : Il est préférable d'enregistrer ou de publier vos pages avant d'y insérer des images. Ceci permet à &brandShortName; d'utiliser automatiquement des références relatives pour les images lorsque vous les insérer.

Pour insérer une image :

  1. Cliquez pour positionner le curseur à l'endroit où vous voulez que l'image apparaisse.
  2. Cliquez sur le bouton Image de la barre d'outils, ou ouvrez le menu Insertion puis choisissez Image…. Vous verrez alors la boîte de dialogue Propriétés de l'image.
  3. Tapez l'adresse et le nom du fichier d'image, ou cliquez sur Parcourir… pour rechercher un fichier d'image sur votre disque dur ou sur le réseau.
  4. Tapez une description succincte de votre image pour le texte alternatif qui apparaîtra à la place de l'image dans les navigateurs en mode caractère (et dans les autres navigateurs aussi) quand une image est chargée ou quand l'affichage des images est désactié.

    Vous pouvez également choisir de ne pas saisir de texte alternatif.

  5. Si besoin, cliquez sur les autres onglets pour ajuster les paramètres (par exemple, l'alignement) dans la boîte de dialogue Propriétés de l'image.

Conseil : Pour insérer rapidement une image : faites un glisser-déposer de celle-ci sur votre page.

Conseil : Pour insérer un saut de ligne sous toutes les images d'un paragraphe, sélectionnez Saut de ligne sous les images dans le menu Insertion.

Éditer les propriétés de l'image

Une fois que vous avez inséré une image dans votre page, vous pouvez éditer ses propriétés et personnaliser l'affichage dans votre page, comme la hauteur, la largeur, l'espacement, et l'alignement du texte. Si vous n'êtes pas en train de visionner la boîte de dialogue Propriétés de l'image, suivez ces étapes :

Pour éditer les propriétés d'une image donnée :

  1. Double-cliquez sur l'image, ou sélectionnez-la et cliquez sur le bouton Image de la barre d'outils pour afficher la boîte de dialogue Propriétés de l'image.
  2. Cliquez sur l'onglet Emplacement pour éditer ces propriétés :
    • Emplacement de l'image : Tapez le nom et le chemin d'accès au fichier image. Cliquez sur Parcourir... pour rechercher une image sur votre disque dur ou sur le réseau.
    • L'URL est relative à l'adresse de la page : Si cette option est cochée, &brandShortName; convertit l'URL pour qu'elle soit relative à l'adresse de la page. Ceci est particulièrement utile si vous prévoyez de publier vos pages sur un serveur Web pour que d'autres personnes puissent les voir. Utiliser des URL relatives vous permet de conserver tous vos fichiers liés au même emplacement relatif les uns par rapport aux autres, sans tenir compte de leur emplacement sur votre disque dur ou sur un serveur Web.

      Si vous décochez cette option, &brandShortName; convertit l'URL en une URL complète (absolue). On utilise généralement des URL absolues pour lier des images présentes sur d'autres serveurs Web (pas pour des images stockées localement sur votre disque dur).

      Si vous n'avez pas encore enregistré ou publié la page, vous devez d'abord l'enregistrer pour activer cette option. (Ce bouton radio n'est pas disponible si vous ouvrez la boîte de dialogue Propriétés de l'image dans une fenêtre de composition de message).

    • Texte alternatif : Entrez un texte qui sera affiché à la place de l'image originale; par exemple, une légende ou une courte description de l'image. C'est une bonne habitude que de spécifier un texte alternatif pour les lecteurs qui utilisent des navigateurs en mode texte ou dont l'affichage des images a été désactivé.
    • Ne pas utiliser de texte alternatif : Choisissez cette option si l'image ne nécessite pas de texte alternatif ou si vous ne voulez pas en inclure.
  3. Cliquez sur l'onglet Dimensions pour éditer ces propriétés :
    • Taille réelle : Sélectionnez cette option pour annuler tous les changements que vous avez fait concernant les dimensions et pour retrouver la taille originale de l'image.
    • Modifier la taille : Sélectionnez cette option et spécifiez la nouvelle hauteur et largeur, en pixels ou en pourcentage de la fenêtre. Ce paramètre n'affecte pas le fichier image original, seulement l'image insérée dans votre page.
    • Conserver les proportions : Si vous changez la taille de l'image, c'est une bonne idée que de choisir cette option pour conserver les proportions de l'image (pour qu'elle n'apparaisse pas déformée). Si vous choisissez cette option, alors vous n'avez besoin de modifier que la hauteur ou la largeur, pas les deux.
  4. Cliquez sur l'onglet Apparence pour éditer ces propriétés :
    • Espacement : Spécifiez la marge entourant l'image; entre l'image et le texte adjacent. Vous pouvez également mettre une bordure noire autour de l'image et spécifier sa largeur en pixels. Mettez zéro pour ne pas avoir de bordure.
    • Placement du texte : Si vous avez placé votre image à côté de texte, sélectionnez une icône d'alignement pour indiquer comment vous voulez que le texte soit placé par rapport à l'image.
    • Image Map : Cliquez sur Supprimer pour supprimer tout paramètre de découpage de l'image.
  5. Cliquez sur l'onglet Lien pour éditer ces propriétés :
    • Entrer l'adresse d'une page Web, d'un fichier local ou sélectionner une ancre ou un titre dans cette liste : Si vous voulez définir un lien pour cette image, entrez l'URL d'une page locale ou distante, sélectionnez une ancre ou un titre à partir de la liste déroulante. Cliquez sur Parcourir… pour chercher un fichier image sur votre disque dur ou sur le réseau.
    • L'URL est relative à l'adresse de la page : Si cette option est cochée, &brandShortName; &brandShortName; convertit l'URL pour qu'elle soit relative à l'adresse de la page. Ceci est particulièrement utile si vous prévoyez de publier vos pages sur un serveur Web pour que d'autres personnes puissent les voir. Utiliser des URL relatives vous permet de conserver tous vos fichiers liés au même emplacement relatif les uns par rapport aux autres, sans tenir compte de leur emplacement sur votre disque dur ou sur un serveur Web.

      Si vous décochez cette option, &brandShortName; convertit l'URL en une URL complète (absolue). On utilise généralement des URL absolues pour lier des images présentes sur d'autres serveurs Web (pas pour des images stockées localement sur votre disque dur).

      Si vous n'avez pas encore enregistré ou publié la page, vous devez d'abord l'enregistrer pour activer cette option. (Ce bouton radio n'est pas disponible si vous ouvrez la boîte de dialogue Propriétés de l'image dans une fenêtre de composition de message).

    • Encadrer les images qui portent un lien : Si cette option est cochée, affiche la couleur de surbrillance utilisée pour les liens autour de l'image.
  6. Pour appliquer des attributs supplémentaires ou des événements JavaScript, cliquez sur Édition avancée… pour afficher l'Éditeur des propriétés avancées.
  7. Cliquez sur OK pour confirmer vos changements.
