Ajouter des tableaux à votre page Web

Insérer un tableau

Les tableaux sont utiles pour organiser du texte, des images, et des données dans des lignes et des colonnes formatées. Pour insérer un tableau :

  1. Cliquez pour positionner le curseur à l'endroit où vous voulez que le tableau apparaisse.
  2. Cliquez sur le bouton Tableau sur la Barre d'outils. La boîte Insérer un tableau apparaît.
  3. Tapez le nombre de lignes et de colonnes que vous voulez.
    • (Optionel) Entrez une taille pour la largeur du tableau, et sélectionnez soit % de la fenêtre soit pixels.
  4. Entrez un nombre pour l'épaisseur de la bordure (en pixels); entrez zéro pour une bordure nulle.

    Note : &brandShortName; utilise une ligne de points rouges pour indiquer un tableau avec une bordure nulle; la ligne de points disparaît quand la page est visionnée dans un navigateur.

  5. Pour appliquer des attributs de tableau supplémentaires ou des événements JavaScript, cliquez sur Édition avancée… pour afficher l'Éditeur des propriétés avancées.
  6. Cliquez sur OK pour confirmer vos paramètres et voir votre nouveau tableau.

Pour modifier des propriétés supplémentaires de votre nouveau tableau, voir Changer les propriétés d'un tableau.

Conseil : Pour insérer un tableau dans un autre tableau, ouvrez le menu Insertion puis choisissez Tableau….

Changer les propriétés d'un tableau

Cette section décrit comment modifier les propriétés qui s'appliquent à l'intégralité du tableau, tout autant que pour les lignes, les colonnes ou les cellules individuelles d'un tableau. Si vous n'êtes pas déjà dans la boîte de dialogue Propriétés du tableau, suivez les étapes suivantes :

  1. Sélectionnez le tableau, ou cliquez n'importe où à l'intérieur.
  2. Cliquez sur le bouton Tableau sur la barre d'outils, ou ouvrez le menu Tableau et choisissez Propriétés du tableau…. La boîte de dialogue Propriétés du tableau contient deux onglets : Tableau et Cellules.
  3. Cliquez sur l'onglet Tableau pour éditer ses propriétés :
    • Taille : Utilisez ceci pour spécifier le nombre de lignes et de colonnes. Entrez la largeur du tableau puis choisissez % de fenêtre ou pixels. Si vous choisissez la largeur comme un pourcentage, la largeur du tableau changera chaque fois que la fenêtre de &brandShortName; ou du navigateur changera.
    • Bordures et marges : Utilisez ceci pour spécifier, en pixels, la largeur de la bordure, l'espace entre les cellules, et l'espacement (l'espace entre le contenu de la cellule et sa bordure).

      Note : &brandShortName; utilise une ligne de points pour afficher les tableaux ayant une bordure nulle; la ligne de points disparaît quand la page est visionnée dans un navigateur.

    • Alignement du tableau : Utilisez ceci pour aligner le tableau dans la page. Choisissez une option dans la liste déroulante.
    • Légende : Choisissez la position de la légende dans la liste déroulante.
    • Couleur de fond : Utilisez ceci pour choisir une couleur pour le fond du tableau, ou pour le laisser transparent.
  4. Pour appliquer des attributs supplémentaires ou des événements JavaScript, cliquez sur Édition avancée… pour afficher l'Éditeur des propriétés avancées.
  5. Cliquez sur Appliquer pour voir vos changements sans fermer la boîte de dialogue, ou cliquez sur OK pour les confirmer.

Pour voir, modifier ou ajouter des propriétés à une ou plusieurs cellules :

  1. Sélectionnez la ligne, la colonne ou la cellule, puis ouvrez le menu Tableau et choisissez Propriétés du tableau…. La boîte de dialogue Propriétés du tableau apparaît.
  2. Cliquez sur l'onglet Cellules pour éditer les propriétés suivantes :
    • Sélection : Choisissez Cellule, Ligne ou Colonne dans la liste déroulante. Cliquez sur Précédente ou Suivante pour vous déplacer dans les lignes, les colonnes ou les cellules.
    • Taille : Taper un nombre pour Hauteur et Largeur, puis choisissez % du tableau ou pixels.
    • Alignement du contenu : Sélectionnez un type d'alignement vertical et horizontal pour le texte ou les données à l'inérieur de chaque cellule.
    • Style de cellule : Sélectionnez Titre dans la liste déroulante pour les titres de colonnes ou de lignes (ce qui centre et met en gras le texte dans la cellule); sinon, choisissez Normal.
    • Retour automatique à la ligne : Sélectionnez Non dans la liste déroulante pour garder le texte sur une même ligne à moins d'insérer un saut de paragraphe. Sinon, choisissez Oui.
    • Couleur de fond : Sélectionnez une couleur pour le fond de la cellule ou laissez-la transparente.

      Note : Pour appliquer des attributs supplémentaires ou des événements JavaScript, cliquez sur Édition avancée… pour afficher l'Éditeur de propriétés avancées

  3. Cliquez sur Appliquer pour voir vos changements sans fermer la boîte de dialogue ou cliquez sur OK pour les confirmer.

Conseil : Pour changer la couleur du texte ou la couleur du fond d'une ou plusieurs cellules sélectionnées ou du tableau entier, sélectionnez les cellules ou cliquez n'importe où dans le tableau puis cliquez sur l'icône de couleur de texte ou de couleur de fond dans la barre de formatage.

Conseil : Pour changer la couleur des cellules avec la dernière couleur utilisée, sélectionnez la cellule, puis appuyez sur la touche Maj et cliquez sur le sélecteur de couleur de fond. Ceci est utile lorsque vous voulez utiliser une couleur pour des cellules individuelles.

Ajouter et supprimer des lignes, des colonnes et des cellules

&brandShortName; vous permet d'ajouter ou de supprimer rapidement une ou plusieurs cellules, colonnes ou lignes d'un tableau. De plus, vous pouvez définir des options qui vous permettent de conserver la structure rectangulaire originale ou l'affichage du tableau lorsque vous réalisez des tâches d'édition.

Pour ajouter une cellule, ligne ou colonne à votre tableau :

  1. Cliquez à l'intérieur du tableau dans lequel vous voulez ajouter une (ou des) cellules.
  2. Ouvrez le menu Tableau et choisissez Insérer.
  3. Choisissez l'un des groupements de cellules. (Vous pouvez également insérer un nouveau tableau dans une cellule).

Pour supprimer une cellule, ligne ou colonne :

  1. Cliquez sur une ligne, colonne ou cellule pour positionner le curseur, ou sélectionnez des cellules adjacentes pour supprimer plus d'une ligne à la fois. Pour sélectionnez des cellules adjacentes, maintenez la touche gauche de votre souris enfoncée et sélectionnez les cellules. Pour sélectionner des cellules individuellement, maintenez la touche Ctrl enfoncée (Windows, Linux ou Unix) ou la touche Commande (Mac OS) et cliquez sur les cellules désirées.
  2. Ouvrez le menu Tableau et choisissez Supprimer.
  3. Puis choisir l'item que vous voulez supprimer.

Pour fusionner une cellule avec celle placée à sa droite :

  • Cliquez dans la cellule de gauche, ouvrez le menu Tableau et choisissez Fusionner avec la cellule suivante.

Pour fusionner des cellules adjacentes :

  • Sélectionnez les cellules adjacentes que vous désirez.
  • Ouvrez le menu Tableau et choisissez Fusionner les cellules sélectionnées.

Pour scinder des cellules précédemment fusionnées :

  • Cliquez dans la cellule fusionnée, ouvrez le menu Tableau, puis choisissez Scinder la cellule. &brandShortName; mettra tout le contenu de la cellule fusionnée dans la première des deux cellules.

Voir Sélectionner des éléments de tableau pour des informations sur la manière de sélectionner des cellules, lignes ou colonnes non-adjacentes.

Changer le comportement par défaut pour l'édition de tableau

Par défaut, quand vous supprimez une ou plusieurs cellules, &brandShortName; préserve la structure de la table en ajoutant les cellules à la fin d'une ligne, quand c'est nécessaire. Ceci vous permet de supprimer une ou plusieurs cellules tout en maintenant l'affichage rectangulaire original ou la structure du tableau. Sinon, supprimer des cellules peut donner un tableau avec des espaces vides, ou dont les bordures apparaîtraient irrégulières à cause des cellules manquantes.

Pour changer le comportement par défaut pour l'édition de tableau, commencez à partir de la fenêtre de &brandShortName; :

  1. Ouvrez le menu Outils (Menu Mozilla sur Mac OS X), choisissez Préférences…, puis Général.
  2. Sous Édition des tableaux, définissez la préférence suivante :
    • Assurez-vous que Conserver les dimensions du tableau lors de l'insertion ou la suppression de cellules soit coché pour éviter d'obtenir un tableau de forme irrégulière.
  3. Cliquez sur OK.

Voir également Définir les préférences générales de &brandShortName;.

Sélectionner des éléments de tableau

Vous pouvez utiliser une des deux manières pour sélectionner rapidement un tableau, une cellule ou un groupe de cellules :

  • Cliquez sur un tableau, ouvrez le menu Tableau, puis le menu Sélectionner puis choisissez un item du sous-menu. Par exemple, pour sélectionner un tableau, cliquez n'importe où dans le tableau, ouvrez le menu Tableau, puis Sélectionner et enfin, choisissez Tableau.
  • Ou vous pouvez utiliser la souris comme outil de sélection :
    • Pour sélectionner un groupe de cellules adjacentes : cliquez sur une cellule puis déplacer le curseur pour sélectionner les cellules que vous voulez. Déplacer la souris vers la gauche ou la droite (en maintenant le bouton gauche de la souris enfoncé) pour sélectionner une ligne; vers le haut ou vers le bas, pour sélectionner une colonne.
    • Pour sélectionner des cellules non adjacentes : appuyez sur la touche Ctrl (Windows, Linux ou Unix) ou Commande (Mac OS) puis cliquez à sur une cellule. Maintenez la touche Ctrl (Windows, Linux ou Unix) ou Commande (Mac OS) enfoncée lorsque vous cliquez pour ajouter des cellules supplémentaires.
    • Pour étendre une sélection pour y inclure des cellules adjacentes : cliquez sur une cellule puis déplacez la souris (en maintenant le bouton gauche enfoncé) sur les cellules supplémentaires pour étendre la sélection.
    • Pour sélectionner une ou plusieurs lignes ou colonnes adjacentes : Sélectionnez avec la souris la première colonne ou ligne, puis, tout en maintenant le bouton gauche de la souris enfoncé, déplacez la souris vers la gauche ou la droite pour sélectionner des colonnes ou lignes supplémentaires. Maintenez la touche Maj enfoncée puis cliquez sur la cellule à l'extrême droite pour sélectionner toute la ligne. Pour sélectionner toute une colonne, procédez de même et cliquez sur la première ou la dernière cellule de la colonne.

Déplacer, copier et supprimer des tableaux

Pour déplacer un tableau :

  1. Cliquez dans le tableau.
  2. Ouvrez le menu Tableau, choisissez Sélectionner puis Tableau.
  • Pour copier ou déplacer un tableau : Utilisez les options Couper, Copier et Coller du menu Edition.
  • Pour supprimer un tableau : Ouvrez encore le menu Tableau, choisissez Supprimer puis Tableau.

Convertir un texte en tableau

Pour convertir un texte en tableau :

  1. Sélectionnez le texte que vous voulez convertir en tableau. Gardez à l'esprit que &brandShortName; crée une nouvelle ligne de tableau pour chaque paragraphe de la sélection.
  2. Ouvrez le menu Tableau et choisissez Créer un tableau à partir de la sélection. Vous verrez alors la boîte de dialogue Convertir en tableau.
  3. Choisissez le caractère que &brandShortName; utilisera pour séparer la sélection en colonnes, ou spécifiez un caractère différent à utiliser. Si vous choisissez Espace comme séparateur de colonne, choisissez si vous voulez ou non que &brandShortName; ignore les espaces multiples et les traite comme une seule espace.
  4. Laissez Supprimer caractère séparateur coché si vous voulez que &brandShortName; supprime le caractère séparateur quand il convertit le texte en tableau. Si vous ne voulez pas que &brandShortName; supprime ce caractère, décochez cette option.
  5. Cliquez sur OK.

Note : Le formatage du texte est supprimé quand le texte sélectionné est converti en tableau.
