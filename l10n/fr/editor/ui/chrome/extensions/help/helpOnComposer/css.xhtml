Utiliser des feuilles de style pour vos pages Web

Utiliser des feuilles de style en cascade

Les feuilles de style en cascade (CSS) peuvent être utilisées pour mettre en forme un document HTML de trois manières :

  1. En utilisant des styles intégrés.
  2. En utilisant une feuille de style interne.
  3. En utilisant des feuilles de styles externes.

&brandShortName; a un éditeur de feuilles de styles intégré appelé CaScadeS. CaScadeS peut être utilisé pour produire soit des feuilles de styles internes, soit externes. Contrairement aux styles intégrés, les feuilles de styles internes ou externes permettent de garder les informations de contenu et de style séparées.

Pour mettre en forme le document HTML que vous éditez, CaScadeS peut être démarré en cliquant sur le menu Outils puis Éditeur CSS… CaScadeS permet deux types de modes d'édition des feuilles de styles :

  1. Mode débutant : Ce mode permet de créer des règles aux sélecteurs de classes ou de type d'éléments.
  2. Mode expert : Ce mode permet de créer des règles sans aucune restriction.

S'il n'y a pas de feuille de styles, une nouvelle sera automatiquement créée.

Créer des feuilles de styles avec &brandShortName;

En utilisant CaScadeS, vous pouvez créer soit une feuille de style interne, soit une feuille de style externe. Pour créer une feuille de style interne :

  1. Cliquez sur le bouton Feuille incorporée
    • (Optionnel) Vous pouvez renseigner les champs Liste des média et Titre.
  2. Cliquez sur le bouton Créer la feuille de style.

Pour créer une feuille de style externe :

  1. Cliquez sur le bouton Feuille liée
  2. Tapez l'URL de la feuille de style dans la partie de droite. Si le fichier n'existe pas, &brandShortName; en créera un nouveau sur votre disque dur.
    • (Optionnel) Vous pouvez renseigner les champs Liste des média et Titre.
  3. Si vous voulez créer une feuille de style alternative, cochez la case cochez pour créer une feuille de style alternative.

Conseil : enregistrez toujours votre document avant d'attacher une feuille de style locale.

Conseil : utilisez le bouton Recharger si la feuille de style n'est pas téléchargée immédiatement.

Créer des règles pour vos feuilles de style

Après avoir créé une ou plusieurs feuilles de style pour votre document HTML, vous pouvez ajouter des règles à chacune individuellement. Pour sélectionner dans quelle feuille de style vous allez créer et modifier des règles, cliquez avec le bouton gauche de la souris sur la feuille concernée dans la partie gauche. Les détails sur la feuille de style s'afficheront dans l'onglet Général de la partie de droite. Pour créer de nouvelles règles :

  1. Cliquez sur le bouton Ràgle de la partie de gauche.
  2. La partie de droite affiche les types de règles que vous pouvez créer. Choisissez parmi :
    • style nommé (entrez un nom de classe ci-dessous)
    • style appliqué à tous les éléments d'un type (entrez le type ci-dessous)
    • style appliqué à tous les éléments correspondant au sélecteur ci-dessous
  3. Saisissez le nom de la règle.
  4. Cliquez sur le bouton Créer la règle de style.

Vous pouvez définir les règles en utilisant les onglets de mise en forme (Texte, Fond, Bordures, Boîte, audio) dans la partie de droite. Pour voir toutes les définitions d'une régle de style, sélectionnez la règle dans la partie de gauche et cliquez sur l'onglet Général de la partie de droite. L'onglet Général affiche toutes les définitions de la règle.
