Définir les propriétés d'une page

Définir les propriétés d'une page et les balises Meta

Utilisez la boîte de dialogue Propriétés de la page pour définir des propriétés tels que le titre, l'auteur et la description du document sur lequel vous êtes en train de travailler. Ces informations sont utiles si vous prévoyez d'utiliser la page sur un site Web, puisque les moteurs de recherche utilisent ce type d'informations pour indexer votre page. Vous pouvez voir ces informations à partir de la fenêtre de votre navigateur en ouvrant le menu Edition puis Informations sur la page.

  1. Ouvrez le menu Format et choisissez Titre et propriétés de la page….
  2. Modifiez l'une des propriétés suivantes :
    • Titre : Tapez le texte que vous voulez voir apparaître comme titre de la fenêtre quand quelqu'un visionne la page avec un navigateur. C'est la façon dont la plupart des moteurs de recherche localisent les pages Web, aussi, choisissez un titre qui corresponde au contenu de la page.
    • Auteur : Tapez le nom de la personne qui a créé le document. Cette information est utile pour les lecteurs qui localisent le document en utilisant un outil de recherche sur le Web qui s'appuie sur le nom de l'auteur.

      Conseil : Si vous avez saisi le nom de l'auteur dans les Préférences de &brandShortName;, vous n'aurez plus alors à le saisir à chaque fois que vous créez une nouvelle page.

    • Description : Entrez une brève description sur le contenu du document.

Définir les couleurs de page et de fond

Vous pouvez changer la couleur de fond ou spécifier une image de fond pour la page sur laquelle vous êtes en train de travailler. Ces choix affectent la façon dont le texte et les liens de votre page apparaissent aux personnes qui visionnent votre page à l'aide d'un navigateur.

Pour définir les couleurs et le fond de page pour la page en cours, dans la fenêtre de &brandShortName; :

  1. Ouvrez le menu Format et choisissez Couleurs et fond de page….
  2. Éditez l'une des propriétés suivantes :
    • Couleurs par défaut (celles du navigateur) : Sélectionnez ceci si vous voulez que votre page utilise les paramètres de couleur du navigateur du lecteur pour le texte et les liens.
    • Couleurs personnalisées : Sélectionnez ceci si vous voulez spécifier les couleurs pour le texte et les liens. Pour chaque élément, sélectionnez une couleur à partir de la liste déroulante. Un exemple de rendu pour chaque type de lien apparaît dans le panneau de droite.
    • Image de fond : Sélectionnez ceci si vous voulez que le fond de votre page soit une image. Tapez le nom du fichier image ou cliquez sur Parcourir… pour chercher l'image sur votre disque dur ou sur le réseau.

      Note : Les images de fond sont empilées et recouvrent donc les sélections pour la couleur de fond.

    • L'URL est relative à l'adresse de la page : Si cette option est cochée, &brandShortName; convertit l'URL pour qu'elle soit relative à l'adresse de la page. Ceci est particulièrement utile si vous prévoyez de publier vos pages sur un serveur Web pour que d'autres personnes puissent les voir. Utiliser des URL relatives vous permet de conserver tous vos fichiers liés au même emplacement relatif les uns par rapport aux autres, sans tenir compte de leur emplacement sur votre disque dur ou sur un serveur Web.

      Si vous décochez cette option, &brandShortName; convertir l'URL en une URL complète (absolue). On utilise généralement des URL absolues pour lier des images présentes sur d'autres serveurs Web (pas pour des images stockées localement sur votre disque dur).

      Si vous n'avez pas encore enregistré ou publié la page, vous devez d'abord l'enregistrer pour activer cette option.

Note : Pour appliquer des attributs supplémentaires ou des événements JavaScript, cliquez sur Édition avancée… pour afficher l'Éditeur des propriétés avancées.

Vous pouvez également définir la page de fond et les couleurs par défaut pour chaque nouvelle page que vous créez avec &brandShortName;.
