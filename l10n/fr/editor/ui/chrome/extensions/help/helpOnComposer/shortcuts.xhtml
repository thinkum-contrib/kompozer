Raccourcis clavier de &brandShortName;

Utiliser les raccourcis

Ce document utilise le format suivant pour lister les raccourcis :

Exemple :

Commande WindowsMac OSUnix/Linux
Copier Ctrl+CCmd+C

Pour accomplir une commande, appuyez sur la touche accélératrice CtrlCmd et appuyez sur la touche combinée de raccourci C une fois (tout en gardant la touche accélératrice enfoncée).

Astuce : Le signe plus (+) signifie que vous devez appuyer sur les deux touches entourant le signe plus en même temps. S'il y a plus d'un signe plus (tel que Ctrl+Maj+C), cela signifie que les trois touches de part et d'autre du signe plus doivent être enfoncées en même temps.

Abréviations :

  • CtrlCmd = Touche de contrôleTouche Commande sur le clavier Mac OS

Note : Certains raccourcis clavier réalisent des fonctions différentes basées sur la position du curseur (le focus). Par exemple, si vous appuyez sur la touche Début lorsque vous êtes en train de visionner une page Web, &brandShortName; se déplacera vers le haut de la page. Cependant, si vous appuyez sur la touche Début alors que le curseur se trouve dans un champ texte, le curseur se placera au début du champ texte.
