Travailler avec des modèles

Les modèles HTML sont utiles quand les auteurs doivent créer plusieurs pages ayant une conception similaire mais un contenu différent. Avec &brandShortName;, vous pouvez concevoir vos propres modèles et les utiliser comme base pour vos pages Web.

Créer un modèle

  1. Pour créer une modèle :
    • Cliquez sur Fichier > Nouveau, ou cliquez sur la flèche sur le côté du bouton Nouveau et choisissez Plus d'options…
    • Sélectionnez Modèle vide et cliquez sur Ouvrir
  2. Ajouter et formater le contenu qui restera identique dans chaque page en utilisant les mêmes méthodes utilisées pour modifier du texte dans une page Web.
  3. Pour le contenu qui sera différent dans les pages Web, une zone éditable peut-être insérée en cliquant sur Insertion > Modèles > Insérer une zone éditable. Les options disponibles pour une zone éditable sont :
    • La zone est optionnelle
    • La zone peut être répétée
    • La zone est déplaçable
  4. Enregistrez le fichier en tant que fichier de modèle .mzt.

Conseil : Vous pouvez supprimer une zone éditable ou rendre une zone éditable en utilisant la barre d'état. Pour les détails, voir Utiliser la barre d'état pour formater vos pages Web.

Utiliser un modèle

Pour utiliser un modèle pour créer des pages Web :

  1. Cliquez sur Fichier > Ouvrir ou utilisez le raccourci clavier Ctrl+N
  2. Sélectionnez l'option Nouveau document utilisant un modèle et choisissez le modèle à utiliser en cliquant sur Parcourir….
  3. Cliquez sur Ouvrir
  4. Modifiez le contenu des zones éditables.
  5. Enregistrez le fichier en tant que document html.
