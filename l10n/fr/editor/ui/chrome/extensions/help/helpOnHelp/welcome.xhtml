Utiliser la fenêtre d'Aide

Cette section décrit comment utiliser la fenêtre d'aide (c'est-à-dire la fenêtre contenant ce texte ainsi que les onglets situés à gauche).

Trouver une rubrique

Les onglets situés à gauche de la fenêtre d'Aide permettent de sélectionner le mode de consultation que vous désirez :

  • L'onglet &toctab.label; affiche la liste des rubriques principales.

    Cliquez sur le titre d'une rubrique pour la consulter. Double-cliquez pour accéder à ses sous-rubriques.

    Si vous ne trouvez pas la rubrique qui vous intéresse, essayez l'onglet &searchtab.label; ou Index.

  • L'onglet &searchtab.label; vous permet d'effectuer une recherche au sein de l'Aide.

    Saisissez un mot ou une phrase et appuyez sur la touche Entrée ou sur le bouton &searchbtn.label; pour voir les rubriques connexes. Cliquez ensuite sur la rubrique qui vous intéresse.

    Si votre recherche n'aboutit pas, essayez en proposant moins de mots ou une combinaison différente.

  • L'onglet &indextab.label; affiche la liste des rubriques dans l'ordre alphabétique.

    Cliquez sur l'entrée d'index que vous souhaitez consulter.

  • L'onglet &glossarytab.label; affiche des définitions dans l'ordre alphabétique.

    Cliquez sur un terme du glossaire pour voir sa définition.

Les liens qui figurent dans l'Aide s'utilisent comme ceux d'une page Web.

Naviguer dans l'Aide

Pour naviguer dans l'Aide, utilisez les boutons de la barre de navigation de la fenêtre d'Aide :

  • Cliquez sur le bouton &sidebarBtn.label; pour ouvrir ou fermer celle-ci à la gauche de la fenêtre d'Aide.
  • Cliquez sur le bouton &backButton.label; pour retourner sur les pages que vous avez visitées précédemment. Il fonctionne comme le bouton &backButton.label; de &brandShortName;.
  • Cliquez sur le bouton &forwardButton.label; pour aller sur les pages suivantes que vous avez visitées précédemment. Il fonctionne comme le bouton &forwardButton.label; de &brandShortName;.
  • Cliquez le bouton &homeButton.label; pour revenir à la page d'accueil de l'Aide, incluant des liens vers des options de support et des ressources Web.

Vous pouvez aussi imprimer les instructions que vous souhaitez suivre :

  • Pour imprimer l'intégralité de la page de l'Aide affichée, cliquez le bouton &printButton.label; de la barre de navigation de la fenêtre d'Aide :
  • Pour imprimer uniquement une partie de la page, sélectionnez la zone que vous souhaitez imprimer avec la souris, puis cliquez le bouton &printButton.label;.

Astuces de recherche dans l'Aide

Vous ne trouvez pas ce que vous cherchez dans la recherche de l'Aide ? Voici quelques astuces qui pourraient vous aider :

  • Assurez-vous de chercher quelque chose qui se rapporte à &brandShortName;. Il ne s'agit pas d'une recherche sur le Web.
  • Essayez d'élargir votre recherche, ne la restreignez pas trop. La formulation de l'Aide pourrait être différente de ce que vous cherchez.
  • Évitez d'être trop large dans votre recherche. Des mots tels &brandShortName; ou &vendorShortName; pourraient retourner toutes les pages de l'Aide.
