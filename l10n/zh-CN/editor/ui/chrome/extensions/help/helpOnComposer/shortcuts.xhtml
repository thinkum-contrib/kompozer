&brandShortName; 键盘快捷键

使用快捷键

本文档用下列格式显示快捷键:

样例:

命令 WindowsMac OSUnix/Linux
复制 Ctrl+CCmd+C

要执行一个命令，按组合键CtrlCmd 和 C(按住控制键不松，再按一下字母键)。

小提示: 加号(+)意思是您要同时按住两个键，如果有两个以上加号 (比如 Ctrl+Shift+C)，意思是同时要按住多个键。

缩写:

  • CtrlCmd = Control 键Mac OS 键盘上的 Command 键

注意: 有些快捷键执行的功能取决于光标所在位置(焦点)。 例如，如果您在查看一个网页时按 Home 键，&brandShortName; 将跳至页面开头， 如果您在一个文本框里按 Home 键，光标将移到文本框的开头。
