使用模板

HTML 模板在设计外观相同，只是内容不同的多个页面时，非常有用。 在 &brandShortName; 中，您可以设计自己的模板，使用它们作为一个页面的基础。

创建模板

  1. 要创建模板:
    • 点菜单 文件 > 新建，或者点工具栏新建按钮旁边的箭头，选“更多选项”
    • 选择 空模板 然后点 "创建”按钮
  2. 就像普通页面一样，添加所有页面相同的内容并设置好格式。
  3. 对于各个页面不同的内容部分，可以通过点菜单 插入 > 模板 > 可编辑区域， 创建可编辑区域。可可编辑区域的选项有:
    • 普通区域
    • 可重复区域
    • 可移动区域
  4. 保存文件为 .mzt 模板文件。

小提示: 你可以使用状态栏删除或更改可编辑区域。更多细节请参阅: 使用状态栏格式化您的网页。

使用模板

用模板制作网页:

  1. 点菜单 文件 > 新建 或者按快捷键“Ctrl+N”
  2. 选择“套用模板的新文档"，然后点“浏览”按钮选择一个模板
  3. 点 创建
  4. 在可编辑区域内输入内容。
  5. 保存文件为一个 html 文档。

小提示: 点击 编辑 > 从模板中分离 使文档从它套用的模板中分离出来。

小提示: 转换一个现有的文档为模板页，可以点菜单 格式 > 页面标题和属性，选中“模板页”选项，反之亦然。
