用目录表格设计您的网页

向网页中添加目录表格

对于一个 html 文档有不同的章节，例如，一个文章 有章和节，每一章和节都有标题， 在单独一页里访问所有内容变得十分困难。 要快速访问不同的章节，在 &brandShortName; 中使用目录表(TOC)，使它变得容易。 在 html 文档中生成一个目录表 TOC:

  1. 使用 html 标题标记(h1, h2, h3, h4, h5, h6)来标记文档中不同级别的章节，然后保存文档。
  2. 点菜单 插入 > 目录表，然后点“插入”菜单，出来一个目录表 TOC 窗口。
  3. 为文档中不同章节标题选择标题级别。
  4. 点击“确定”按钮创建目录表 TOC。

默认创建的目录表用数字显示所有项，分别用锚链接到各自的章节。 不选中“编号表中的所有项目”复选框使用符号样式，选中“设定目录表只读“ 目录表的文字成为简单的文本，没有锚链接。

更新或删除目录表

在创建目录表后，文档可能会改变，添加更多的章节，包含新的章或节， 要添加新的章节，点菜单“插入－目录表”，然后选择更新命令，目录表将会更新从而显示新的项目。

删除目录表，点菜单“插入－目录表”，然后选删除命令，目录表将会被删除，html 源代码会自动清理添加的 TOC 代码。
