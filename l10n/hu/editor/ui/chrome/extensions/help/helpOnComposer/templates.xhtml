Sablonok kezelése

A HTML-sablonok olyan esetekben hasznosak, amikor a szerzőnek több oldalt kell elkészítenie hasonló designnal, e eltérő tartalommal. Az &brandShortName; segítségével megtervezhetjük saját sablonjainkat, amelyek a weboldalaink alapját képezhetik.

Sablon létrehozása

  1. Sablon létrehozása:
    • Kattintsunk a File > New menüparancsra, vagy kattintsunk az "Új" gomb melletti nyílra, és válasszuk a "További lehetőségek" lehetőséget.
    • Válasszuk az Üres sablon lehetőséget, majd kattintsunk a "Létrehozás" gombra
  2. Szúrd be és formázd a tartalomat ami azonos marad minden oldalon, ugyanazzal a metódussal mint amivel normál szöveget formázol szerkesztő módban.
  3. Azokat a tartalmakt, amik különbözőképpen jelennek meg az oldalakon, beszúrhtod a következő képpen: kattints: Insert > Templates > Editable Area. Itt a követező opciók lelhetők fel:
    • A terület opcionális
    • A terület ismételhető
    • A terület áthelyezhető
  4. Mentsd el .mzt Template álloménynak.

Tipp: El lehet távolítani egy elemet a készenléti bár segítségével. Részletekért kattints ide.

Sablonk használata

Egy template használata:

  1. Kattints a File > Open-re vagy nyomd le a "Ctrl+N"-t.
  2. Válaszd ki ezt "A New Document based on a Template" és a "Choose File" segítségével válaszd ki a használni kivánt templatet.g
  3. Kattints a Create-re
  4. Szerkeszd a tartalmat ahol szükséges.
  5. Mentsd el html dokumentumként.

Tipp: Kattints az Edit > Detach From Template hogy detachálj egy dokumentumot a templateből.

Tipp: Egy dokumentum transzformálása templatebe és vissza meg lehet oldani "Templates" beixelésével a Format > Page Title and Properties menübeng.
