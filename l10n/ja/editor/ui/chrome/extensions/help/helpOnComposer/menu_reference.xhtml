&brandShortName;メニューリファレンス

このメニューリファレンスでは、&brandShortName;のメインウィンドウの各メニューの項目を解説します。

ファイル

新規作成

新しい文書を&brandShortName;の新しいタブ、または新しいウィンドウで開きます。

ファイルを開く…

ディスク中のファイルを選択するダイアログを表示します。&brandShortName;を利用して、HTML、HTMLの雛型、テキストファイルを表示することが出来ます。

URIを指定して開く…

文書のURLを指定してその文書の編集を開始します。またローカルファイルも指定できます。

最近使用したページ

最近編集したファイルの一覧を表示します。

閉じる

現在のウィンドウを閉じます。

すべて閉じる

現在のウィンドウのすべてのタブを閉じます。

保存

現在選択されているタブのファイルを保存します。

名前を付けて保存…

現在編集中のページに別の名前を付けて保存します。

文字エンコードを指定して保存…

ページの文字エンコーディングを指定するダイアログを先に表示して、現在編集中のページを「名前を付けて保存」します。

公開

現在編集中のページを公開 (FTPにアップロードなどする) する、公開ダイアログを表示します。

名前を付けて公開…

現在編集中のページを公開 (FTPにアップロードなどする) する、公開ダイアログを表示します。「公開」と同じ動作です。

変更の破棄

現在編集中のページに加えられた変更を破棄し、直前に保存した内容まで巻き戻します。

画像の再読み込み

現在編集中のページ内に埋め込まれた画像のみを再読み込みします。

テキストとして出力…

現在の文書の内容をプレーンテキストとして出力します。

ページをプレビュー

現在編集中のページを標準で関連付けられたウェブブラウザで表示します。

印刷ページ設定…

印刷ページ設定ダイアログを表示して、印刷時の余白、ヘッダ、フッタ、印刷方向の指定が出来るようにします。

印刷プレビュー

現在表示中のページを印刷するとどのようになるのかのプレビュー画面を表示します。印刷ページ設定ダイアログの設定が反映されます。

印刷…

何枚印刷するかなどを設定する、印刷ダイアログを表示します。OKをクリックするとページを印刷します。

終了

&brandShortName;のすべてのウィンドウを閉じ、現在継続中のダウンロードを中止して、&brandShortName;を完全に終了します。もし複数のタブが開いたままになっていれば、それを閉じてよいか確認するダイアログが表示されます。誤って終了操作をしても安心です。

編集

元に戻す

現在編集中の文書に対して行った最後の操作を取り消します。この操作自体を取り消したいときは、「やり直し」コマンドを使用してください。

やり直し

最後の「元に戻す」コマンドを取り消します。

切り取り

現在編集中の文書の選択文字列を削除し、その内容をクリップボードに保存します。「貼り付け」コマンドでその文字列をどこにでも挿入することが出来ます。

コピー

選択文字列の内容をクリップボードに保存します。「貼り付け」コマンドでその文字列をどこにでも挿入することが出来ます。

貼り付け

クリップボード内の文字列を、カーソル位置に貼り付けます。

書式情報を省いて貼り付け

クリップボード内の文字列を単純な文字列として、カーソル位置に貼り付けます。つまり、書式に関する情報は省略されます。

削除

現在編集中の文書の選択文字列を削除します。

すべて選択

現在編集中の文書のすべての文字列を選択します。

検索・置換…

文字列を検索したり、他の文字列に置き換えることができる、ダイアログを表示します。この検索は現在の文書に対して行われます。

次を検索

検索・置換コマンドで指定した文字列の次に該当する部分検索します。この検索は現在の文書に対して行われます。

前を検索

検索・置換コマンドで指定した文字列の前に該当する部分検索します。

綴りをチェック

現在編集中のページの綴りを確認する為のダイアログを表示します。

雛型と区別する

「雛型」を編集した文書を、htmlとして保存できるように区別するようにします。

公開サイト設定

サイトマネージャを開き、公開対象のサイトを指定、設定を変更します。

表示

表示切替え

編集ツールバー

アンカー、リンク、画像など、編集時に用いるボタンがある編集ツールバーの表示・非表示を切り替えます。

書式ツールバー

文字色、文字揃えなど、書式を整える時に用いるボタンがある、書式ツールバーの表示・非表示を切り替えます。

編集モードツールバー

編集モードツールバーの表示・非表示を切り替えます。編集モードツールバーとは、&brandShortName;最下段にあるステータスバーの上のバーのことです。HTMLタグ表示、HTMLソース表示などの編集モードを切り替えることが出来ます。

ステータスバー

&brandShortName;ウィンドウの一番下にある、ステータスバーの表示・非表示を切り替えます。

物差し

編集画面の上、左に表示される物差しの表示・非表示を切り替えます。

ツールバーのカスタマイズ

メインツールバー

編集ツールバーのカスタマイズパネルを開きます。ボタンなどの部品をパネルからツールバーにドラッグアンドドロップすることで新しいボタンを追加したり、逆に取り外したり出来ます。

書式ツールバー

書式ツールバーのカスタマイズパネルを開きます。ボタンなどの部品をパネルからツールバーにドラッグアンドドロップすることで新しいボタンを追加したり、逆に取り外したり出来ます。

サイトマネージャ

&brandShortName;ウィンドウのサイトマネージャの表示・非表示を切り替えます。

通常の編集モード

通常の編集モードへと表示モードを切り替えます。

HTMLタグ

HTMLタグ表示モードへと切り替えます。

HTMLソース

HTMLソース表示モードへと切り替えます。

プレビュー

プレビューモードへと切り替えます。

ブロックの輪郭

ページ中のブロックレベル要素の輪郭線の表示・非表示を切り替えます。

文字サイズ

小さく

ページ中の文字を小さくします。同様の操作はマウスやキーボードショートカットを使っても出来ます (次項参照)。元通りにするには、Ctrl+0を押します。

大きく

ページ中の文字を大きくします。同様の操作はCtrlキーを押しながらマウスホイールを動かしたり、+や-キーを押すことでも出来ます。元通りにするには、Ctrl+0を押します。

文字エンコード

ウェブページの文字エンコードを手動で変更することが出来ます。

挿入

現在編集中の文書に、様々なHTML要素を追加します。

画像…

現在の文書に画像を挿入するダイアログを開きます。

テーブル…

現在の文書にテーブルを挿入するダイアログを開きます。

入力フォーム

リンク…

現在の文書にリンクを作成する、ダイアログを開きます。文字列選択時にこのコマンドを実行すると、選択文字列をリンクとします。また、詳細表示から、リンクの関連性を定義するXFN (XHTML Friends Network) などを定義することが出来ます。

名前付きアンカー…

現在のカーソル位置に、名前付きアンカーを挿入するダイアログを開きます。

水平線

現在の文書に水平線 (<hr>) を挿入します。

HTML…

HTMLを直接入力するダイアログを開きます。

文字・シンボル…

特殊文字や記号などを選択するダイアログを開きます。選択された文字はカーソル位置に挿入されます。

目次…

目次を作成し、現在のカーソル位置に挿入します。目次は<h1>～<h6>、<div>、<p>タグから作成することが出来ます。

ページ部品

ページの構成要素となる部品を挿入します。

'Made With &brandShortName;'ボタン

"Made with &brandShortName;" を挿入します。

カレンダー

動的に生成される、カレンダーを挿入します。

雛型

「雛型」(テンプレート) を作成時に、このコマンドで編集可能領域を追加することが出来ます。

PHPコード

PHPコードをHTML文書に挿入します。

コメント

HTMLのコメントをHTML文書に挿入します。

画像の下で改行

「画像の下で改行」を追加します。具体的には、<br clear="all">を挿入します。

書式

文書の書式を整えるコマンドがあります。

フォント

現在編集中の文書で選択されている文字列の、フォントフェイスを選択します。システムにインストール済みのすべてのフォントなどから選択することが出来ます。

サイズ

1段階小さく (smaller)、1段階大きく (larger)で、現在表示されている文字の大きさから相対的に変更します。

とても小さい (xx-small)、小さい (x-small)、ちょっと小さい (small)、通常 (medium)、ちょっと大きい (large)、大きい (x-large)、とても大きい (xx-large)を指定すると、それに相当する大きさに変更します

文字装飾

現在編集中の文書で選択されている文字列の、HTMLの文字スタイルを選択することが出来ます。

ボールド体 (太字)

ボールド体で表示する指定を追加します。一般的には「太字」とされているものです。例：ボールド体

イタリック体 (斜体)

イタリック体で表示する指定を追加します。例：イタリック

下線付き

選択文字列に下線を付けます。例：下線付き

打ち消し線付き

選択文字列に重なる横線を付けます。例：打ち消し線付き

上付き文字

選択文字列の文字の大きさを少し小さくして、通常文字より位置を少し上にします。例：上付き文字

下付き文字

選択文字列の文字の大きさを少し小さくして、通常文字より位置を少し下にします。例：上付き文字

固定幅

選択文字列のフォントフェイスを等幅フォント (monospace) にします。例：固定幅

途中で改行しない

HTMLを表示する際、連続する空白は一つに纏められ、その位置で改行されることがあります。その位置で改行して欲しくない場合には、空白を選択してこのコマンドを実行してください。例：   この前に3つの「改行禁止の空白」があります。

強調 (em)

選択文字列を「強調」します。HTMLでの<em>です。例：強調

より強調 (strong)

選択文字列を「より強調」します。HTMLでの<strong>です。例：より強調

引用元 (cite)

選択文字列を「引用元」を表すものとしてマークアップします。HTMLでの<cite>です。例：引用元

略語 (abbr)

選択文字列を「略語」を表すものとしてマークアップします。HTMLでの<abbr>です。例：UN (United Nations: 国連)

頭字語 (acronym)

選択文字列を「頭字語」を表すものとしてマークアップします。HTMLでの<acronym>です。例：WWW (World Wide Web)

プログラムコード (code)

選択文字列を「プログラムコード」を表すものとしてマークアップします。HTMLでの<code>です。例：プログラムコードの断片

出力例 (samp)

選択文字列をコンピュータの「出力例」を表すものとしてマークアップします。HTMLでの<samp>です。例：プログラムが出力するデータの例

変数 (var)

選択文字列を「プログラム変数」を表すものとしてマークアップします。HTMLでの<var>です。例：変数foo

文字色…

Opens the color picker popup window to choose the color for selected text.

色選択ダイアログを開き、選択文字列の文字色を変更します。

文字方向

選択文字列の文字描画方向を指定します。方向は左から右へ (英語など)、右から左へ (アラビア語など) からせん宅で着ます。この描画方向は、インラインレベル要素ではなくブロックレベル要素に指定されねばなりません。

すべての装飾を解除

現在のカーソル位置で、すべての装飾を解除します。

リンクを解除

現在のカーソル位置の前までをリンクとし、以降をリンクから外します。

名前付きアンカーを削除

現在選択されている、名前付きアンカーを削除します。

段落

選択文字列の属するブロックレベル要素を、body直下 (注：Strict DTDでは不正)、段落、H1～H6、引用、連絡先、Preから選択します。

リスト

一覧としてマークアップするものが選択できます。順番付き (<ol>)、順番なし (<ul>)、定義項目 (<dt>)、定義内容 (<dd>)、そしてリストのプロパティが選択可能です。

インデントを1段階深くする

ブロックレベル要素にインデントを表すスタイルを付けます。もし既に付いていればそれをさらに深くします。

インデントを1段階浅くする

ブロックレベル要素のインデントを表すスタイルを浅い値にします。

配置

選択文字列を含むブロックレベル要素の配置を決定します。左、右、中央、そして均等割り (justify)が選択可能です。

グリッド配置

レイヤーを移動する際の解像度を設定するダイアログを開きます。

詳細プロパティ

「詳細プロパティ」ダイアログを開きます。このダイアログから、HTML属性、インラインスタイル、JavaScriptイベントの追加・編集が出来ます。

ページの配色と背景…

ページの色、背景画像を選択するダイアログを開きます。本文、リンク文字列、選択されたリンク文字列、訪問済みリンク文字列の色、背景の色と背景画像が選択可能です。

ページのタイトルとプロパティ…

「ページのプロパティ」ダイアログを開きます。このダイアログから、ページのタイトル、著作者、内容の説明、ページ内容が記述されている言語、文字方向が指定できます。もし現在編集中のページを「雛型」(テンプレート) として用いるのであれば、このページを「雛型」として用いるにチェックを入れてください。

選択

このセルを右と結合

選択文字列からテーブルを作成
