&brandShortName;のキーボードショートカット

ショートカットの使い方

ここでは、次のような形でショートカットの一覧を掲載しています。

例：

命令 WindowsMac OSUNIX/Linux
コピー Ctrl+CCmd+C

コマンドを実行するには、一緒に表示されているボタンを同時に押してください。 例えば、コピーを行うには、CtrlCmd キーとCキーを同時に押します。

ヒント: プラス (+) 記号は、その左右に書かれたキーを同時に 推すことを意味します。複数のプラス記号がある場合、例えばCtrl+ Shift+Cのような場合は、記号の前後に書かれた3つのキーを 同時に押してください。

凡例：

  • CtrlCmd = コントロールキーMac OS用キーボードのコマンドキー

注意: キーボードショートカットの中には、カーソルの位置 (フォーカスされている場所) によって異なる機能が割り当てられているものがあります。 一例を挙げると、Windowsの場合、ウェブページを閲覧中にHomeキーを 押すと、ウェブページの先頭に移動します。これがカーソルがテキスト入力欄の時は、 カーソルの位置が入力欄の先頭に移動します。
