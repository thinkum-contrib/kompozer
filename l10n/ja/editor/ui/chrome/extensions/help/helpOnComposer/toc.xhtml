ウェブページの目次を作る

ウェブページの目次を追加する

複数の章や節からなるHTML文書を一つのファイルにすると、その内容の案内が大変になってしまいます。各章・節の案内を簡単にする為に、&brandShortName;には目次を簡単に作成する機能が備わっています。HTML文書の目次を作成するには次のようにします。

  1. HTMLの見出しタグ (h1、h2、h3、h4、h5、h6) を用いて、章や節の深さ毎に区別をします。そして文書を保存します。
  2. 挿入 → 目次をクリックし、挿入を選択します。「目次」ウィンドウが表示されます。
  3. 章・節を表すのに用いた見出しのレベルを選択します。
  4. OKをクリックすると見出しが生成されます。

通常はすべての目次の項目が番号付きで生成され、対応する章・節へのアンカーがつきます。「目次中のすべての項目に番号を振る」のチェックを外すと、目次を順番なしの一覧として生成します。「目次を編集不可にする」と、目次にアンカーを生成せず文字列のみにします。

目次の更新・削除する

目次を作成してから文書に章・節を追加しても、追加されたものは目次には載っていません。新しく追加された章・節を目次に追加するには、挿入 → 目次をクリックし、更新を選択します。新しい項目が追加されます。

目次を削除するには、挿入 → 目次をクリックし、削除を選択します。目次は削除され、目次作成に伴い追加されたHTMLコードは削除されます。
