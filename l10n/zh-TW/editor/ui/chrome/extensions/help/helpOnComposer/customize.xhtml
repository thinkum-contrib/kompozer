自訂工具列

&brandShortName; 的工具列能讓使用者依照自己的喜好而調整。使用者可以將他們最常用的按鍵保留起來，而將少用的按鍵隱藏起來，並且依自己的喜好來調整 &brandShortName; 的外觀。

如何調整工具列

要開啟自訂工具列視窗，請點擊「檢視」─「顯示/隱藏」─「自訂工具列」，並選擇您要調整的工具列。或者您也可以在您要編輯的工具列上點擊滑鼠右鍵，並且選擇「自訂工具列」。

自訂工具列

  • 您可以點擊並拖曳一個按鍵、分隔器、彈性空間或空間項目到工具列上的位置。
  • 您可以在自訂工具列視窗中將項目（按鍵、空間、彈性空間或分隔器）從工具列中拖出來，或拖回工具列中。
  • 您也可以變更工具列按鍵（主工具列）的外觀，選擇顯示圖示與文字，或只顯示圖示或文字等。

加強 &brandShortName; 的功能

&brandShortName; 有一個內建的機制，可以增強它的功能。此外您也可以用擴充套件來新增功能。擴充套件通常是支小程式，安裝在 &brandShortName; 內可以增加它的功能。使用者可以選擇要安裝的擴充套件。因此擴充套件可以讓使用者自訂 &brandShortName;。要安裝擴充套件：

  • 下載擴充套件到您的電腦上，並儲存起來。大部份的擴充套件都可以在 &brandShortName; 網站上取得。
  • 在「工具」選單中點擊「擴充套件」，即可開啟擴充套件管理員（EM）。EM 會列出所有已安裝的擴充套件。
  • 點擊「安裝」鍵即可安裝您下載的擴充套件。
  • 在檔案系統瀏覽視窗中選擇擴充套件，並點擊「開啟」。
  • 在軟體安裝視窗中點擊「現在安裝」。
  • 安裝完成後，重新啟動 &brandShortName; 即可使用新增的功能。

您應該定期用擴充套件管理員檢查安裝的擴充套件是否有更新。通常更新可以改進功能，並解決前一版的問題。
