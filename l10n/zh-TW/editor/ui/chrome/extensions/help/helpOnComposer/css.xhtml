使用樣式表來指定您網頁的格式

使用 CSS 樣式表

CSS（Cascade Style Sheets）可以用三種方式來指定 HTML 文件的格式：

  1. 使用內含樣式。
  2. 使用內部樣式表。
  3. 使用外部樣式表。

&brandShortName; 有一個內建的樣式表編輯器，叫做 CaScadeS。CaScadeS 可以用於產生內部或外部的樣式表。與內含樣式表不同，內部或外部樣式表可以將內容與樣式資訊分離開來。

要在您編輯的 html 文件中加入樣式，您可以點擊「工具」並選擇「CSS 編輯器」。CaScadeS 可以用兩種模式來編輯樣式表：

  1. 初學者模式：允許建立與類別或元素型態選擇器的相關規則。
  2. 專家模式：允許建立任何規則，沒有限制。

若是沒有樣式表，它會自動建立一個新的樣式表。

用 &brandShortName; 建立樣式表

使用 CaScadeS，您可以建立內部或外部樣式表。要建立內部樣式表的話：

  1. 點擊「樣式」按鍵
    • （選擇性的）填入媒體清單與樣式表的標題資訊。
  2. 點擊以建立樣式表。

要建立外部樣式表：

  1. 點擊「連結」按鍵
  2. 在右方填入樣式表的網址。若是有需要，它會在本地端檔案系統內建立新檔案。
  3. （選擇性的）填入媒體清單與樣式表的標題資訊。
  4. 若有替代樣式表，請勾選「勾選以建立替代樣式表」。

提示：在附加本地樣式表前，請先儲存 html 文件。另外，在關閉 CSS 編輯器前，也要立刻儲存文件。

提示：若未自動下載，您可以用「刷新」按鍵來下載樣式表。

建立樣式表規則

建立一個以上的樣式表後，您可以建立個別樣式表的規則。要建立或變更樣式表的規則，先在左方用滑鼠左鍵點擊樣式表，然後在右方「一般」分頁中會顯示出樣式表詳情。此時要建立新規則：

  1. 點擊左邊「規則」按鍵
  2. 右方會根據要建立哪種規則而顯示選項。請從中間選取一個：
    • 已命名樣式（在下方輸入類別名稱）
    • 套用到所有特定型態的元素的樣式（在下方輸入型態）
    • 套用到所有符合下列選擇器的元素的樣式
  3. 填入規則的名稱
  4. 點擊「建立樣式規則」按鍵。

您可以用右方的樣式分頁（文字、背景、邊框、排版定位與語音）來定義規則。要查看所有樣式規則的定義，請先在左方選取樣式，並點擊右方的「一般」分頁。「一般」分頁中會顯示出規則目前套用的定義。
