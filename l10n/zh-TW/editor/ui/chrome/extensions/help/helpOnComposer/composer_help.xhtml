用 &brandShortName; 建立網頁

&brandShortName; 讓您輕鬆建立與發佈自己的網頁。您不需要瞭解 HTML，就像用一般文字處理程式一樣就行了。

工具列按鍵讓您新增列表、表格、影像、連結、顏色與字型。您可以直接看到您的網頁的樣子，並且與其他使用者分享您的文件，不管他們用的是哪種瀏覽器或電子郵件程式。

開始新頁面

建立新頁面

&brandShortName; 是一個 HTML 編輯器，讓您可以建立與編輯網頁。&brandShortName; 是一個所見即所得（WYSIWYG）的編輯器，在編輯過程中，您看到的頁面就會是您的讀者所看到的。您不需要瞭解 HTML，因為大部份 HTML 的功能都可以由工具列與選單中取得。

&brandShortName; also lets you edit the HTML source if you want. To view or edit the HTML source code, open the View menu, and choose HTML Source, or click the <HTML> Source tab in the Edit Mode toolbar at the bottom of the &brandShortName; window.

要建立頁面，只要用以下其中一種方法。您開始編輯一個頁面之後，您可以像在文字編輯器裡一樣新增與編輯文字

在 &brandShortName; 裡建立新頁面:

  • 在 &brandShortName; 工具列中點擊「新增」按鍵。

要開啟您本地的 HTML 檔:

  1. 開啟視窗選單，並選擇 &brandShortName;。您就可以看到 &brandShortName; 視窗。
  2. 開啟檔案選單並選擇開啟檔案。您可以看到「開啟 HTML 檔」的對話框。
  3. 請指定您要編輯的本地檔案。
  4. 點擊開啟在 &brandShortName; 視窗中顯示您指定的檔案。

要編輯網頁:

  1. 開啟 &brandShortName;
  2. 點擊檔案選單並選擇開啟網頁位置。您可以看到「開啟網頁位置」對話框。
  3. 輸入要編輯的網頁的網址（例如，http://www.mozilla.org）並點擊「建立」按鍵。您會看到網頁顯示在 &brandShortName;。

&brandShortName; 也會給您一些用 XHTML建立新文件的選項。要建立 XHTML 文件，請在「建立新文件或樣本」的對話框中勾選「建立 XHTML 文件」。

提示：在 &brandShortName; 視窗中您可以快速開啟最近使用過的檔案，只要在「檔案」選單中，選擇「最近開啟的頁面」，並選擇您要使用的檔案。

分頁─編輯多個文件

&brandShortName; 讓您可以同時編輯多份文件。您不需要開多個視窗，只要使用「分頁」的功能。您可以同時在 &brandShortName; 視窗中將多個文件開啟在不同的分頁中。

分頁可以用 Ctrl+T 或在開啟新檔案對話框中選擇「建立於：新分頁」。雖然在不同的分頁，不過彼此之間是獨立的。每個文件的狀態都會呈現出來。

儲存與瀏覽您的新頁面

您可以用 HTML 或純文字格式儲存您的 &brandShortName; 文件。以 HTML 格式儲存，會保留文件的格式，如文字樣式（粗體或斜體），表格，連結，圖片等。以純文字模式儲存的話，會移除所有 HTML 標籤，但保留文字的部份。

要將文件儲存成 HTML 檔：

  • 開啟「檔案」選單，選擇「儲存」或直接點擊工具列上的「儲存」按鍵。

    若您尚未指定頁面的標題，&brandShortName; 會提示您。&brandShortName; 會將頁面標題顯示在瀏覽器視窗的標題列。文件的頁面標題也會在您將頁面加入書籤時，出現在書籤的清單中。

    &brandShortName; 接著會提示您輸入檔名與指定儲存檔案的位置。請確定您保留 .html 做為副檔名。儲存或編輯 XHTML 文件時，副檔名應為 .xhtml。以 .html 做副檔名的 XHTML 文件並不會被認為是 xhtml 文件。

要變更 HTML 檔的檔名或位置：

  • 選擇「檔案」─「另存為」，並輸入新檔名或位置。

當您在 &brandShortName; 中儲存頁面時，頁面中的所有部件（HTML，影像與其它檔案，如音效等、樣式表等），都會儲存在您的硬碟中。若您只要儲存 HTML 的部份，您必須變更 &brandShortName; 的儲存頁面的喜好設定。關於變更儲存頁面的設定，請參考 &brandShortName; 喜好設定。

若是影像位置為絕對位置（以 http:// 開頭），同時您已連線到網際網路，您可以在 &brandShortName; 與瀏覽器中看見該圖片。不過，若是相對位置（以 file/// 開頭），則您將無法在本地端文件看到圖片。

要將文件存成純文字檔：

  1. 開啟「檔案」選單，並選擇「匯出到文字檔」。
  2. 輸入檔名並指定您要儲存檔案的位置。

注意：影像不會出現在純文字格式的文件中。

提示：您可以選擇「檔案」選單中的「回復」，取回最近一次儲存的文件複本。不過這樣一來，您目前的變更將會消失。

在瀏覽器中檢視您的頁面並測試連結：

  • 開啟「檔案」選單，選擇「瀏覽頁面」，或是在工具列上點擊「瀏覽」。若您尚未儲存您的文件，&brandShortName; 會提示您輸入頁面標題、檔名與位置。在瀏覽視窗開啟後 &brandShortName; 視窗並不會消失。
