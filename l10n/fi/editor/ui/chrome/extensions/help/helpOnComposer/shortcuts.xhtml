&brandShortName;:n näppäinkomennot

Näppäinkomentojen käyttö

Tämä dokumentti luetteloi näppäinkomennot seuraavassa muodossa:

Esimerkki:

Toiminto WindowsMac OSUnix/Linux
Kopioi Ctrl+CCmd+C

Suorita komento painamalla toimintonäppäin CtrlCmd pohjaan ja paina näppäinkomennon toista näppäintä C kerran, jonka jälkeen voit päästää toimintonäppäimen irti.

Vinkki: Plus-merkki (+) tarkoittaa, että merkkiä ympäröiviä näppäimiä tulee painaa samaan aikaan. Jos plus-merkkejä on enemmän kuin yksi (esimerkiksi Ctrl+Shift+C), tulee kaikkia plus-merkkejä ympäröiviä näppäimiä painaa samaan aikaan.

Lyhenteitä:

  • CtrlCmd = Control-näppäin Command-näppäin Mac OS:n näppäimistössä

Huomaa: Jotkut näppäinkomennot suorittavat eri komentoja riippuen kohdistimen sijainnista. Jos esimerkiksi painat Home-näppäintä Windowsissa tarkastellessasi www-sivua, &brandShortName; siirtyy sivun yläreunaan. Jos painat Home-näppäintä Windowsissa kun kohdistin on tekstikentässä, kohdistin siirtyy tekstikentän alkuun.
