Tastaturkürzel benutzen

Im Folgenden wird folgendes Format zur Auflistung der Tastaturkürzel benutzt:

Beispiel:

Befehl Windows Mac OS Linux oder Unix
Kopieren Strg+C Cmd+C Strg+C

Um den jeweiligen Befehl auszuführen, drücken Sie die angegebenen Tasten gleichzeitig. Zum Beispiel, um unter Windows etwas zu kopieren, drücken Sie gleichzeit Strg und C.

Tipp: Das Plus-Zeichen (+) bedeutet, dass Sie die beiden angegebenen Tasten gleichzeitig drücken müssen. Wenn mehr als ein Plus-Zeichen angegeben ist (wie zum Beispiel Strg+Shift+C) müssen Sie alle drei Tasten gleichzeitig drücken.

Abkürzungen:

  • Strg = Steuerungstaste
  • Cmd = Command-Taste auf der Mac OS- Tastatur

Anmerkung: Manche Tastaturkürzel haben verschiedene Funktionen, abhängig von der Psotion des Cursors (Focus). Zum Beispiel, drücken Sie unter Windows "Pos1" während Sie eine Webseite anschauen, so wird &brandShortName; an den Anfang der Webseite springen. Drücken Sie "Pos1" jedoch wenn sich der Cursor in einem Textfeld befindet, so springt der Cursor an den Anfang des Textfeldes.
