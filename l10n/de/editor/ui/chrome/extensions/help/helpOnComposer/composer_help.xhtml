Erstellen einer neuen Seite

Erstellen einer neuen Seite

&brandShortName; ist ein HTML (Hypertext Markup Language)-Editor mit dem Sie Webseiten erstellen und bearbeiten können. &brandShortName; ist ein WYSIWYG (What You See Is What You Get)-Editor, so dass Sie direkt bei der Erstellung sehen können wie die Seite später im Web verfügbar sein wird. Sie müssen dafür keine HTML-Kenntnisse besitzen, da die grundlegenden Funktionen über Schaltflächen und Menüs verfügbar sind.

&brandShortName; lässt Sie trotz allem auch selber den Quelltext ändern. Um den Quelltext bearbeiten zu können, klicken Sie im Menü "Ansicht" auf "Quelltext" oder klicken Sie auf den <HTML>Quelltext-Tab unten in der Editiermodus-Symbolleiste.

Um eine Webseite zu erstellen, verwenden Sie eine der unten beschriebenen Möglichkeiten. Haben Sie erstmal eine Seite erstellt, so können Sie Text hinzufügen und ändern wie Sie es von einem Textverarbeitungsprogramm gewöhnt sind.

Um eine neue Seite in &brandShortName; zu erstellen:

  • Klicken Sie auf die Schaltfläche "Neu" in der Symbolleiste.

Um eine lokal existierende Datei zu bearbeiten:

  1. Öffnen Sie &brandShortName;.
  2. Klicken Sie auf "Datei" und wählen dort "Datei öffnen". Sie sehen die Dialog-Box "HTML-Datei öffnen".
  3. Wählen Sie die zu ändernde Datei aus.
  4. Klicken Sie auf "Öffnen". Die Datei öffnet sich nun in einem &brandShortName;-Fenster.

Um eine Webseite im Web zu bearbeiten:

  1. Öffnen Sie &brandShortName;
  2. Klicken Sie auf "Datei" und wählen dort "Webadresse öffnen".
  3. Geben Sie die URL der zu bearbeitenden Seite ein (zum Beispiel, http://www.mozilla.org) und klicken dann auf "Erstellen". Sie sehen nun die angegebene Seite in &brandShortName;.

Tipp: In &brandShortName; können Sie ganz einfach die letzten bearbeiteten Dateien aufrufen. Öffnen Sie dazu im Menü den Eintrag "Datei". Dort wählen Sie den Eintrag "Letzte geladene Seiten" aus und können nun die gewünschte Seite auswählen.

Speichern und Ansehen Ihrer neuen Seite

Sie können &brandShortName;-Dokumente im HTML- oder Textformat speichern. Im HTML-Format bleiben alle Formatierungen erhalten, wie Textstile (zum Beispiel fett oder kursiv, Tabellen, Links und Grafiken. Speichern Sie das Dokument hingegen im Textformat werden alle HTML-Tags entfernt und nur der Text gespeichert.

Um ein Dokument als HTML-Datei zu speichern:

  • Klicken Sie auf "Datei" und wählen Sie "Speichern" oder klicken Sie auf die "Speichern"-Schaltfläche in der Symbolleiste.

    Wenn Sie Ihrer Seite noch keinen Titel gegeben haben, fordert &brandShortName; Sie nun dazu auf. &brandShortName; zeigt den Titel in der Titeleiste des Browsers an wenn die Seite darin geöffnet wird. Der Titel wird ebenso in den Lesezeichen angezeigt.

    &brandShortName; fordert Sie danach auf, den Dateinamen und Speicherort auszuwählen. Beachten Sie, dass die ".html" als Endung ausgewählt haben.

Um den Dateinamen oder den Speicherort einer vorhandenen Datei zu ändern:

  • Wählen Sie "Speichern unter" und geben einen anderen Dateinamen oder Speicherort an.

Wenn Sie eine Seite in &brandShortName; speichern werden alle Teile der Seite (der HTML-Code, die Grafiken und andere Dateien, wie Sounddateien und Style Sheets), auf Ihrer Festplatte gespeichert. Möchten Sie jedoch nur den HTML-Code speichern, so müssen die &brandShortName;-Einstellungen für neue Seiten ändern. Für weitere Informationen siehe &brandShortName;-Einstellungen.

Ist die Adresse einer Grafik absolut angegeben (beginnend mit "http://") und eine Internetverbindung besteht, so wird die Grafik im Dokument von &brandShortName; und Ihrem Browser angezeigt. Ist die Adresse einer Grafik jedoch relativ angegeben (beginnend mit "file:///"), so wird die Grafik in Ihren lokalen Dokument nicht angezeigt.

Um eine Datei im Textformat zu speichern:

  1. Klicken Sie auf "Datei" und wählen "Als Text exportieren" aus.
  2. Geben Sie den Dateinamen ein und wählen den Speicherort aus.

Anmerkung: Grafiken erscheinen nicht in Dateien, die im Textformat gespeichert werden.

Tipp: Mit dem Befehl "Zurück" im Menü "Datei" können Sie die zuletzt gespeicherte Version Ihrer Seite wieder aufrufen. Beachten Sie, dass alle Änderungen die Sie danach durchgeführt haben, verloren gehen.

Um Ihre Seite in Ihrem Browser anzuzeigen (zum Beispiel, um Ihre Links zu testen):

  • Im Menü "Datei" wählen Sie die Option "Vorschau" aus oder klicken in der Symbolleiste auf "Vorschau". Haben Sie Ihre Seite noch nicht gespeichert, So wird &brandShortName; Sie nun dazu auffordern, den Seitentitel, Dateinamen und Speicherort anzugeben. Das &brandShortName;-Fenster bleibt im Hintergrund geöffnet.

Formatieren Ihrer Webseiten

Formatieren von Absätzen, Überschriften und Listen

Um einem Absatz Formate zuzuweisen:

  1. Klicken Sie ins Fenster um den Cursor an die Position zu setzen, an dem die Formatierungen beginnen sollen oder markieren Sie einen Textbereich.
  2. Wählen Sie ein Absatzformat aus. Dies geschieht über das Drop-Down-Menü in der Format-Symbolleiste:
    • Normaler Text: Dem Text werden die Standardschriftart und der Standardschriftschnitt zugewiesen.
    • Absatz: Fügt einen Absatz-Tag ein (einer neuer Absatz wird eröffnet. Der Absatz beinhaltet einen oberen und unteren Rand.
    • Überschrift 1 - Überschrift 6: Formatiert den Absatz als Überschrift. Überschrift 1 ist die höchste Kategorie und Überschrift 6 die niedrigste Kategorie.
    • Adresse: Adresse kann als Signatur einer Webseite verwnedet werden. Es wird der Autor der Seite genannt, sowie eine Person für weitere Informationen, zum Beispiel: user@example.com

      Vielleicht möchten Sie auch das Datum und einen Copyright-Vermerk einfügen. Diese Formatierung erscheint normalerweise am Ende einer Webseite unter einer horizontalen Linie. Der Browser stellt diesen Bereich kursiv dar.

    • Vorformatiert: Diese Formatierung ist besonders nützlich für Code-Beispiele, Datenreihen und Emails, die in einer proportionalen Schriftart dargestellt werden sollen. In normalem Text entfernen die meisten Browser zusätzliche Leerzeichen, Tabulatoren und Absatzmarken. Text mit der Formatierung "Vorformatiert" wird ohne jegliche Änderungen genauso wie der Originaltext dargestellt.

Einen Text als Überschrift formatieren:

  1. Klicken innerhalb des Textes den Sie als Überschrift formatieren möchten.
  2. Mit Hilfe des Drop-Down-Menüs in der Format-Symbolleiste wählen Sie die gewünschte Kategorie der Überschrift aus, von 1 (größte) bis 6 (kleinste). Benutzen Sie "Überschrift 1" für Ihre Hauptüberschrift. "Überschrift 2" für die nächst kleinere Kategorie usw.

Um eine Liste anzulegen:

  1. Klicken Sie in den Text, den Sie als Liste formatieren möchten.
  2. Öffnen Sie das "Format"-Menü und wählen dort "Liste" aus.
  3. Wählen Sie den Listentyp:
    • Unsortiert: Jeder Eintrag bekommt einen Punkt vorangestellt (wie in dieser Liste).
    • Numeriert: Einträge sind numeriert.
    • Ausdruck und Definition: Diese beiden Typen arbeiten bei der Erstellung eines glossar-ähnlichen Stils zusammen. Benutzen Sie den "Ausdruck"-Tag für den zu definierenden Begriff und den "Definition"-Tag für die Erklärung. Der Ausdruck erscheint links und die Definition wird etwas eingerückt.

Tipp: Sie können einen beliebigen Text in eine Liste umwandeln in dem Sie den Text markieren und dann "numerierte Liste" oder "unsortierte Liste" in der Symbolleiste auswählen.

Um den Stil der Punkte bzw. Nummern zu ändern:

  1. Markieren Sie den entsprechenden Listeneintrag oder wählen Sie mehrere Einträge aus, die Sie ändern möchten.
  2. Öffnen Sie das "Format"-Menü und wählen dann "Listeneigenschaften" aus.
  3. Nun können Sie im Drop-Down-Menü die verschiedenen Stile auswählen. Bei numerierten Listen können Sie die Startnummer angeben. Bei unsortierten Listen können Sie daus Aussehen der Punkte verändern.

Tipp: Ein Doppelklick auf die Punkte oder Zahlen einer Liste ruft ebenfalls die Dialogbox "listeneigenschaften" auf.

Um die Ausrichtung eines Absatzes zu ändern, zum Beispiel zentriert oder links- oder rechtsbündig:

  1. Klicken Sie in den entsprechenden Absatz.
  2. Öffnen Sie das "Format"-Menü und wählen dort "Ausrichtung". Wählen Sie nun die von Ihnen gewünschte Ausrichtung.

Anmerkung: Sie können auch die "Format-Symbolleiste" benutzen, um den Text neu auszurichten.

Mit Listen arbeiten

Um eine Liste zu beenden und normalen Text zu schreiben:

  • Setzen Sie den Cursor ans Ende des letzten Listeneintrags und drücken Sie "Enter" (Return unter Mac OS) zweimal um die Liste zu beenden.

Um einen oder mehrere Listeneinträge in normalen Text umzuwandeln:

  1. Wählen Sie den entsprechenden Listeneintrag aus.
  2. Bei einer numerierten bzw. unsortierten Liste klicken Sie auf die entsprechende Schaltfläche in der "Format-Symbolleiste".

Um einen eingerückten Text in einem Listeneintrag zu verwenden:

  1. Setzen Sie den Cursor in den entsprechenden Listeneintrag.
  2. Drücke Sie Shift-Enter um den Text einzurücken.
  3. Geben Sie den Text ein.
  4. Drücken Sie erneut Shift-Enter für einen weiteren, eingerückten Absatz oder drücken Sie Enter um den nächsten Listeneintrag zu erstellen.

Tipp: Sie können die Einrückung vergrößern oder verkleinern. Dazu klicken Sie auf die entsprechenden Schaltflächen in der "Format-Symbolleiste". Alternativ können Sie zum vergößern des Einzugs die "Tab"-Taste drücken, zum verkleinern "Shift+Tab".

  1. Wählen Sie beide Listen aus. Achten Sie darauf, dass Sie alle Einträge markiert haben. Sollte sich zwischen den beiden Listen normaler Text befinden, so wird dieser ebenfalls in die Liste integriert. Beachten Sie dies.
  2. Klicken Sie nun auf die entsprechenden Schaltflächen in der "Format-Symbolleiste" um die beiden Listen zusammenzufügen.

Ändern der Textfarbe, des Schriftschnitts und der Schriftart

Um den Schriftschnitt, die Schriftart oder die Textfarbe eines markierten Textes zu ändern:

  1. Markieren Sie den entsprechenden Textbereich.
  2. Öffnen Sie das "Format"-Menü und wählen Sie einen der folgenden Einträge aus:
    • Schriftart: Hiermit ändern Sie die Schriftart. Möchten Sie jedoch die Schriftarten des jeweiligen Browsers benutzen, so wählen Sie "variable" bzw. "feste Breite" aus.

      Anmerkung: Nicht alle auf Ihrem Computer installierten Schriftarten erscheinen. Damit Sie nicht eine Schriftart auswählen, die nicht auf jedem Computer vorhanden ist, ist es im Allgemeinen am besten eine Schriftart zu wählen die auf jedem Computer gleich angezeigt wird. Wählen Sie jedoch andere Schriftarten aus, so können diese auf anderen Computern anders aussehen als Sie es möchten.

    • Schriftgrad: Benutzen Sie dieses um eine relative Schriftgröße festzulegen oder vergrößern bzw. verkleinern Sie die Schriftgröße relativ zum umliegenden Text.
    • Schriftschnitt: Benutzen Sie dieses um den Schriftschnitt festzulegen wie kursiv, fett oder unterstrichen.
    • Textfarbe: Benutzen Sie dieses um die Textfarbe auszuwählen. Wenn Sie Kenntnisse der hexadezimalen Farbcodes haben, so können Sie diese direkt eingeben oder Sie geben direkt den Namen der Farbe ein (zum Beispiel "blue".

Um die Hintergrundfarbe der Seite zu ändern:

  1. Klicken Sie auf eine beliebige Stelle innerhalb der Seite.

  3. Wählen Sie eine Hintergrundfarbe in der erscheinenden Dialogbox aus.
  4. Wählen Sie danach "OK".

Tipp: Um die Textfarbe schnell zu ändern, markieren Sie den entsprechenden Text, drücken die "Shift"-Taste und klicken auf die Schaltfläche "Textfarbe auswählen" in der "Format-Symbolleiste". Dies ist sehr nützlich wenn Sie eine Farbe für mehrere verschiedene Textbereiche benutzen möchten.

Sie können aber auch eine Grafik als Hintergrund verwenden. Siehe dazu Seitenfarben und -hintergrund.

Quelltext überprüfen

Bevor Sie Ihr Dokument auf einem Webserver veröffentlichen, sollten Sie die HTML-Formatierung überprüfen, um sicherzustellen, daß es den Webstandards entspricht. Dokumente, die aus gültigem HTML bestehen, werden beim Betrachten mit verschiedenen Webbrowsern weniger Probleme hervorrufen. Eine rein visuelle Kontrolle in Ihrem Browser stellt nicht sicher, daß das Dokument auch von anderen Webbrowsern korrekt angezeigt wird.

&brandShortName; bietet einen komfortablen Weg, um sicherzustellen, daß das Dokument den HTML-Standards des W3C (World Wide Web Consortium) entspricht. Der Composer verwendet den W3C HTML Validation Service, der die HTML-Struktur des Dokuments auf Richtigkeit nach dem HTML 4.01 Standard überprüft und Informationen zur Behebung von Fehlern bietet.

Anmerkung: Um dieses Feature zu nutzen, muß eine Internet-Verbindung bestehen.

Um die HTML-Struktur eines Dokumentes zu überprüfen:

  1. öffnen Sie das Menü "Extras" und wählen Sie Quelltext überprüfen. Wurden seit dem letzten Speichervorgang das Dokument verändert, fragt &brandShortName;, ob man diese vorher speichern möchte, bevor man fortfährt.
  2. &brandShortName; öffnet es weiteres Fenster mit den Ergebnissen der Überprüfung.

Den richtigen Editiermodus wählen

Im Normalfall ist es nicht notwendig, den Standard-Editiermodus zu ändern. Sollten Sie jedoch mit dem HTML-Quellcode eines Dokumentes arbeiten wollen, ist es notwendig, den Editiermodus zu wechseln.

Links erstellen

Links innerhalb derselben Seite erstellen

Um einen Link innerhalb derselben Seite zu erstellen, beispielsweise um dem Leser die Möglichkeit zu geben, zwischen einzelnen Absätzen hin- und her zu springen, muß zuerst ein Anker (Zielort) erstellt werden, und anschließend ein Link, der auf den Anker zeigt. Anker werden auch als Ziele bezeichnet.

  1. Setzen Sie den Cursor an den Beginn einer Zeile, für die Sie einen Anker erstellen wollen oder markieren Sie den Text.
  2. Wählen Sie aus dem Menü Einfügen den Punkt Benanntes Ziel. Der Dialog Ziel-Eigenschaften erscheint.
  3. Geben Sie einen eindeutigen Namen für den Anker in das Feld Ziel-Name ein (bis max. 30 Zeichen). Wenn Sie Leerzeichen eingeben, werden diese automatisch in Unterstriche ( _ ) umgewandelt. Wenn Sie in Schritt 1 bereits Text markiert haben, steht dieser Text bereits im Eingabefeld.
  4. Klicken Sie auf OK. Ein Anker-Symbol erscheint in Ihrem Dokument, um die Position des Ankers anzuzeigen:

Um den Link zu erzeugen, der zum Anker führt:

  1. Markieren Sie den Text oder das Bild, das Sie als Link zum Anker verwenden wollen.
  2. Klicken Sie auf den Button Link in der Symbolleiste oder wählen Sie aus dem Menü Einfügen den Punkt Link. Der Dialog Linkeigenschaften erscheint.
    • Wenn Sie einen Link auf eine andere HTML-Datei auf Ihrem Computer erstellen wollen, klicken Sie auf Durchsuchen und wählen die passende Datei aus.
    • Wenn Sie einen Link auf ein benanntes Ziel (Anker) innerhalb der Seite erstellen wollen, wählen Sie den Anker aus dem Drop-Down-Menü der Anker dieser Seite aus.
    • Wenn Sie einen Link auf eine überschrift (zum Beispiel überschrift 1 bis überschrift 6) einfügen wollen, wählen Sie aus der Liste der überschriften der aktuellen Seite aus.
  3. Klicken Sie OK.

Anmerkung: Um den gerade erstellten Link zu testen, wählen Sie im Menü Datei den Punkt Vorschau. Dort klicken Sie dann auf den Link.

Tipp: Falls Sie noch keine Anker erstellt haben, können Sie zumindest Links auf die überschriften in Ihrer Seite erzeugen.
